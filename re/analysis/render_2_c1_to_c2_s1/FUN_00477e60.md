## Body summary

Computes the plane equation of triangle (A, B, C):
1. Edge vectors: `e1 = A - B`, `e2 = C - B`.
2. Cross product into `param_1[0..2]`:
   - `nx = (Cy-By)*(Az-Bz) - (Cz-Bz)*(Ay-By)`
   - `ny = (Cz-Bz)*(Ax-Bx) - (Cx-Bx)*(Az-Bz)`
   - `nz = (Cx-Bx)*(Ay-By) - (Cy-By)*(Ax-Bx)`
3. Normalizes normal via `FUN_004c39b0(param_1, param_1)`.
4. Computes plane offset: `d = -(n . A)` = `-(nx*Ax + ny*Ay + nz*Az)`.
   Stores to `param_1[3]`.

Output: normalized plane equation `[nx, ny, nz, d]` where `dot(n, P) + d = 0` for any point P on the plane.
