## Mechanical description
- Quaternion SLERP plus linear translation.
- param_1 = output keyframe, param_2 = key A pointer, param_3 = key B pointer, param_4 = blend t.
- Computes dot product fVar8 of two quaternions; negates B if dot < 0.
- Linear translation: `out[6..8] = lerp(A[6..8], B[6..8], t)` at offsets 0x18/0x1c/0x20.
- If `|dot| < threshold`: computes acos via polynomial-Padé approximation (DAT_005d8dcc..0x5d8dec coefficients), divides into two sin-fractions, calls FUN_004c3b30 (FastSqrt) once. Result yields sin(t*theta)/sin(theta) blend weights.
- Else: identity LERP weights (1-t, t).
- Writes blended quaternion to `param_1+0x8..0x14`.
