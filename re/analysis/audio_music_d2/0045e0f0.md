## Mechanical description
- Takes `(channel_index: int, volume: float)`.
- Clamps `volume`:
  - If `volume < DAT_005d757c`: `volume = 0.0f`.
  - Else if `volume > _DAT_005cc320`: `volume = 1.0f`.
  - (decompiler-observed; instruction addresses not independently verified — U-2210)
- Loads audio object pointer: `(&DAT_0068f640)[channel_index * 2]` (stride-8 array, even slots are pointers).
- Writes clamped volume to `*(obj_ptr + 0x40)`.
- If `*(obj_ptr + 0x30) != 0`: calls `FUN_005a6dc0(*(obj_ptr+0x30), 5, 1, volume)` — writes volume to the sub-object via param_type=5.
- Writes clamped volume to `(&DAT_0068f644)[channel_index * 2]` (= `DAT_0068f640 + 4`, the odd/scale slot of the stride-8 array).

## Interpretation
Per-channel volume setter. Sets both the live object field (`obj+0x40`) and the persistent per-channel scale in the stride-8 audio array (`array_base+4+channel*8`). The sub-object call uses `param_type=5` (cf. `param_type=4` used by callers in `004623e0.md`).
