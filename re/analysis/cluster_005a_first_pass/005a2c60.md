## Mechanical description

Comprehensive facet diagnostic printer. Handles three sentinel pointer values:
  - `param_2 == (float*)0x2` → emit `"MERGEridge"` and return.
  - `param_2 == (float*)0x1` → emit `"DUPLICATEridge"` and return.
  - `param_2 == (float*)0x0` → emit `"NULLfacet"` and return.

Otherwise saves `DAT_009146a0 = DAT_00913f8c; DAT_00913f8c = 0` (saves qh.IStracing).

Emits header `" - f%d"` with param_2[0xe] (= facet+0x38 = f.id).
Emits `"    - flags: "`.

Bit-by-bit flag decode of `param_2[0xf]` (= facet+0x3c, qh's f.flags 32-bit word):
  - 0x1000 → `"top" / "bottom"`
  - 0x2000 → `"simplicial"`
  - 0x200 → `"tricoplanar"`
  - 0x20000 → `"upperDelaunay"`
  - 0x800 → `"visible"`
  - 0x400 → `"newfacet"`
  - 0x8000000 → `"tested"`
  - 0x80000 (cleared) → `"notG"`
  - 0x4000 → `"seen"`
  - byte+0x3f & 1 → `"coplanar"`
  - 0x2000000 → `"mergehorizon"`
  - 0x10000000 → `"keepcentrum"`
  - 0x200000 → `"dupridge"`
  - 0x400000 & ~0x800000 → `"mergeridge1"`
  - 0x800000 → `"mergeridge2"`
  - 0x20000000 → `"newmerge"`
  - byte+0x3e & 1 → `"flipped"`
  - 0x40000 → `"notfurthest"`
  - 0x40000000 → `"degenerate"`
  - sign bit → `"redundant"`

Then emits:
  - `"- area: %2.2g"` if flag 0x100000 (hasarea).
  - Else various horizon/replacement/owner messages based on flags 0x400, 0x200, 0x800+visible interaction.
  - `"- merges: %d"` if `param_2[0xf] & 0x1ff` (the merge counter in low 9 bits).
  - Normal: walks param_2[3] (facet+0xc = f.normal), emits per-coord `"%8.4g"`, count = DAT_0091401c.
  - `"- offset: %10.7g"` with param_2[2] (facet+8 = f.offset).
  - If `DAT_009145f8 == 1` OR `param_2[5] != 0.0`: call FUN_005a0e00(param_1, 5, "    - center: ", param_2) to print centroid.
  - `"- maxoutside: %10.7g"` if param_2[1] > DAT_00914464.
  - Outside set walk (`param_2[0xb]`): if size < 6 prints per-point coords; else prints `"- outside set: %d points"` with furthest distance.
  - Same pattern for coplanar set (`param_2[0xc]`).
  - Vertex list: `"   - vertices: p%d(v%d) …"` (point-id, vertex-id).
  - Neighbour facet ids: `"   - neighboring facets: f%d f%d …"`.

Restores `DAT_00913f8c = DAT_009146a0`.
