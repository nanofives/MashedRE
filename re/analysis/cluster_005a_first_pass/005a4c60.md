## Mechanical description

Prints one ridge:
  - `fprintf(param_1, "     - r%d", *(ridge+0xc) & 0xffffff)` (ridge id).
  - Optional `"tested"` if 0x2000000 bit set.
  - Optional `"nonconvex"` if 0x4000000 bit set.
  - Newline.

  - Walks ridge.vertices at *param_2 (+4 chain): per vertex `"p%d(v%d)"` with `qh_pointid(*(v+8))` and `*(v+0x14) & 0xffffff`.
  - Newline.
  - If both top/bottom facets non-null (`param_2[1] != 0 && param_2[2] != 0`): `"     between f%d and f%d"`.
