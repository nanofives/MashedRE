## Mechanical description

Prints one vertex.

If param_2 == 0: emits `"     - NULLvertex"` and returns.

Otherwise:
  - Emits `"     - p%d (v%d):  "` with `qh_pointid(*(v+8))` and `*(v+0x14) & 0xffffff`.
  - Walks vertex coords at *(v+8), DAT_0091401c floats: per-coord `"%5.2g"`.
  - Optional `"deleted"` if (+0x14 & 0x8000000) set.
  - Optional `"ridgedeleted"` if (+0x14 & 0x4000000) set.
  - Newline.
  - If `*(v+0xc) != 0` (vertex.neighbors): emits `"  neighbors: "`, walks neighbours, per: emits `&DAT_00627a78` with neighbour id. Every 100 entries inserts wrap line `s__00626488`. Newline.
