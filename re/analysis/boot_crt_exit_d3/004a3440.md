## Mechanical description
- Takes no formal parameters; reads allocation size from `EAX` register (`in_EAX`).
- If `in_EAX < 0x1000` (4096): writes the return address word at `(SP - in_EAX)` and returns.

- After loop: writes the return address word at `(puVar1 - 4 - in_EAX)` and returns.
