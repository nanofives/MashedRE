## What it does

Allocates and initialises a RenderWare free-list pool (`RwFreeList` equivalent). Called by `FUN_004c8690` to create the shader constant pool.

**Pool header layout (uint[9+]):**
- `[0]`: aligned element size  
- `[1]`: elements per block  
- `[2]`: bitmap size in bytes  
- `[3]`: alignment  
- `[4]`: free-list head ptr  
- `[5]`: free-list tail ptr  
- `[6]`: ownership flag (2=header owned by pool, 3=caller-owned)  
- `[7]`: next ptr in global chain  
- `[8]`: back-ptr to global chain anchor
