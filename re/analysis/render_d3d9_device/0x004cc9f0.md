## What it does

Destroys a free-list pool created by `FUN_004cc820` (`RwFreeListDestroy` equivalent).

1. **Unlinks from global pool chain** at `DAT_007d45cc` (via back/forward pointers at `param_1+0x1c`/`+0x20`)
2. **Walks internal free list** (doubly-linked at `param_1+0x10`): calls `free(+0x10c)` on each block
3. **Frees pool header**: if ownership flag at `param_1+0x18` bit 0 is 0 (we own it), either `free-to-pool` or direct `free` depending on whether `DAT_007d45fc` matches

Called by `FUN_004c8690` (reset path) and `FUN_004c8c70` (teardown).
