## What it does

Same CPUID-support check as `FUN_004dcf90`. Returns true if **either** bit 25 or bit 26 of the DWORD at offset `+8` is set.

`0x6000000 = bits 25 and 26`.

**If `+8` = EDX (most likely for a 2004 game):**
- EDX bit 25 = **SSE**
- EDX bit 26 = **SSE2**
- Function returns true if SSE or SSE2 present.

**If `+8` = ECX:** bit 25 = AESNI, bit 26 = XSAVE — less likely for 2004 game.

**Most likely interpretation:** `has_SSE_or_SSE2()`. See also U-0248 (same uncertainty as FUN_004dcf90).
