## What it does

Same CPUID-support check as `FUN_004dcf90`/`FUN_004dcff0`. Returns bit 26 (0x1a) of the DWORD at offset `+8`.

**If `+8` = EDX (most likely for a 2004 game):** EDX bit 26 = **SSE2** capability.

**Most likely interpretation:** `has_SSE2()`. Criterion RW D3D9 backends use SSE2 for optimised vertex transformation paths.

See U-0248 for struct layout uncertainty shared with FUN_004dcf90 and FUN_004dcff0.
