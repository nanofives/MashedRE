## Mechanical description

Audio timing accumulator tick. Updates a running time counter from the system high-resolution clock.

1. `uVar1 = FUN_005ab040()` — reads current system tick count.
2. `iVar2 = FUN_005ab070()` — reads tick frequency (ticks per second).
3. Computes delta with wraparound: `uVar3 = uVar1 - DAT_007DCA54; if (uVar1 < DAT_007DCA54) uVar3 -= 1`.
4. Updates saved tick: `DAT_007DCA54 = uVar1`.
5. Accumulates: `DAT_007DCA50 += (int)ROUND((float)uVar3 / (float)iVar2 * DAT_005CC9FC)`.

`DAT_005CC9FC` is a fixed-point scale constant; the accumulation converts elapsed ticks to a game-time unit.
