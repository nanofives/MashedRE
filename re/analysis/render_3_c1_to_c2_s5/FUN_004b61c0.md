## Mechanical description

Renders a mesh's edges as a wireframe line-list from indexed triangle data stored in a mesh-data object.

- `param_1` — mesh-data object pointer.
  - `*(param_1+8)` — pointer to vertex array.
  - `*(ushort*)(param_1+0x82)` — vertex count (capped to 0x2000).
  - `*(ushort*)(param_1+0x84)` — face/triangle count (capped to 0x2000).
  - `*(int*)(param_1+4)` — index buffer pointer.
- `param_2` — RGBA color (4 bytes); defaults to `0xFFFFFFFF` (white, opaque) if NULL.
- Alpha blend: if `param_2[3] != -1` → `local_4 = 0x12`, else `0x10`.
- Index construction (global buffer `DAT_007bbe48`):
  - Reads triangle index triples `(a,b,c)` from `*piVar1+4` (stride 8 bytes per index? ushort2-packed), builds 6 line indices per triangle `(a,b,b,c,c,a)` into `DAT_007bbe48`.
- Vertex copy (global buffer `DAT_00773e4c`):
  - Copies vertex XYZ + color into `DAT_00773e4c` with stride 9 dwords per vertex (XYZ + color). Unrolled 4× per outer iteration.
- Calls `FUN_004cd070(&DAT_00773e48, local_10, 0, local_4)` — begin Im3D with vertex count.
- If nonzero: `FUN_004cd170(1, &DAT_007bbe48, iVar11)` — indexed draw, then `FUN_004cd140()`.
- Restores alpha blend state.
- Calls `DAT_007d3ff8+0x20` with `(7,1)/(7,2)` for point-size set/restore.
