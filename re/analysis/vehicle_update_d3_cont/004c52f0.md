**param_3 == 0 (copy):**  
Copies 16 uints from `param_2` → `param_1`; returns `param_1`.

**param_3 == 1 (pre-multiply / replace-with-src):**

- If `param_2[3] & flag` (src is identity): copy `param_1` to tmp → copy tmp back to `param_1`.
