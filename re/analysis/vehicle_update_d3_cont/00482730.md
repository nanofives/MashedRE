# 0x00482730 — PhysicsBodySetMaterialColor

**Size:** 0x0048279d − 0x00482730 = 0x6D bytes

## Mechanical description

Sets RGBA color and lighting scalars on all materials of a RW body's geometry.

1. `iVar1 = *(param_1+0x18)` — geometry.
2. Loop over `*(iVar1+0x24)` materials at `*(iVar1+0x20)`:
   - `*(material+4) = *param_2` — RGBA color (4 bytes packed: R=0x40, G=0x40, B=0x60, A=0xFF = dark blue-grey).
   - `*(material+0x0c) = 0x3f800000` — ambient = 1.0f.
   - `*(material+0x10) = 0x3f800000` — diffuse = 1.0f.
   - `*(material+0x14) = 0x3f000000` — specular = 0.5f.
