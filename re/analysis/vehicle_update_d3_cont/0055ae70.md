# 0x0055ae70 — PhysicsSceneBodySetShapeFlags

**Size:** 0x0055af36 − 0x0055ae70 = 0xC6 bytes

## Mechanical description

Sets shape collision flags for a body slot in the physics scene.

1. `FUN_00565570(*param_1, *(param_2+0x20), param_3)` — apply shape flags to scene slot `*(body+0x20)` with value `param_3`.
2. Check if slot bit in `param_1[0x1a]` (active shapes bitfield) is set for `*(param_2+0x20)`.
3. If active: `FUN_00565200(*param_1, slot, local_20)` — get old shape data; `FUN_00565260(*param_1, slot)` — clear old shape.
4. If old data was retrieved: `FUN_00564c80(*param_1, slot, local_20)` — apply updated shape; set bit in `param_1[0x1a]`.

Called with `iVar6 = 1` (single-player mode) or `iVar6 = vehicleIdx + 2` (multi-player mode 0x26), based on `FUN_00426c00()` return.
