## Mechanical description
- **PointPlaneClassify** — tests which side of plane `param_1` (4 floats) point `param_2` (Vec3) lies on, and optionally writes signed distance to `param_3`.
- `signed_dist = param_1[0]*param_2[0] + param_1[1]*param_2[1] + param_1[2]*param_2[2] + param_1[3]`.
- If `param_3 != NULL`: computes the perpendicular foot of `param_2` on the plane (`param_2 - signed_dist * normal`), then `*param_3 = |param_2 - foot|` (= absolute distance), via Vec3Magnitude FUN_004c3ac0.
- Returns:
  - 0 if `signed_dist > 0` (in front)
  - 1 if `signed_dist == 0` (on plane)
  - 2 if `signed_dist < 0` (behind)
