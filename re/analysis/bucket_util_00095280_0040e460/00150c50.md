## Mechanical description
`undefined4 FUN_00550c50(int param_1 /*fs*/, DWORD *param_2 /*slot*/, undefined4 param_3 /*name*/, byte param_4 /*mode flags*/)` — opens a disk file backing a RtFS slot via CreateFileA, translating mode bits into Win32 access/share/disposition/flags.

1. `uVar1 = FUN_00551410(param_3, 0x5c)` — normalize path separators (0x5c = '\\') in the name.
2. `iVar2 = FUN_00551460(aCStack_100, 0x100, *(fs+0xc) /*mount prefix*/, uVar1)` — build the full path into a 256-byte buffer; if it returns 0 → `return 5` (error).
3. Build CreateFileA params from `param_4`:
   - `(param_4 & 0x10)`: if clear `slot[6]=0`; if set `dwFlagsAndAttributes=0x60000000`, `slot[6]=1`, `slot[8]=3`.
   - `(param_4 & 2)`: if clear (read) `dwCreationDisposition=3` (OPEN_EXISTING), `flags |= 1`; if set (write) `dwShareMode=2`, `access=0x40000000` (GENERIC_WRITE), `disposition = ((param_4&8)!=0 ? 4 : 2)` (OPEN_ALWAYS vs CREATE_ALWAYS), and if `slot[6]==0` set `flags=0x80` (FILE_ATTRIBUTE_NORMAL).
   - `(param_4 & 1)`: `access |= 0x80000000` (GENERIC_READ), `share |= 1` (FILE_SHARE_READ).
4. `hFile = CreateFileA(fullpath, access, share, NULL, disposition, flags, NULL)`; store in `slot[0x10]`.
5. If `hFile == INVALID_HANDLE_VALUE` → `return 3`.
6. Init slot state: if `slot[6]==0` set `slot[8]=2`; `slot[9]=1`; `slot[2]=0`; `slot[3]=0`.
7. `slot[0] = GetFileSize(hFile, slot+1)` (size low into slot[0], high into slot[1]).
8. Save `GetLastError()` into the per-thread errno block (`*(FUN_00550740()+0x10)`).
9. `return 1` (success).
