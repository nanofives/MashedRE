## Body summary
Normalizes a 3-component float vector and returns the inverse-sqrt scale:

1. Computes squared magnitude of `param_2`: `sum = param_2[0]^2 + param_2[1]^2 + param_2[2]^2`.
2. If `sum != 0`: computes `invSqrt` via the RW fast-inverse-sqrt table:  
   `*(*(DAT_007d3ffc + 4 + DAT_007d3ff8)[ (sum+0x800) >> 12 & 0xfff ] * 4) + (~(sum+0x800) >> 1 & 0x3fc00000)`.  
   (Two-level table lookup identical to `FastInvSqrt` at 0x004c3b90.)
3. Writes normalized output: `param_1[0..2] = param_2[0..2] * invSqrt`.
4. Returns `invSqrt` (as float10/extended).

This is `Vec3Normalize` — uses the same FastInvSqrt table as `FUN_004c3b90` to normalize a 3D vector in-place to a separate output buffer.
