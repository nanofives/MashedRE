## Mechanical description

Renders a textured 2D quad on-screen using screen-space UV coordinates. Analogous to a 2D blit / DrawSprite.

Parameters:
- `param_1` — float[2] UV start (top-left), defaults to `{0.0, 0.0}`.
- `param_2` — float[2] UV end (bottom-right), defaults to `{1.0, 1.0}`.
- `param_3` — blend mode / texture index.
- `in_EAX` — RGBA colour (4 bytes), defaults to `{0xff, 0xff, 0xff, 0xff}`.

1. Defaults: `local_80 = {0,0,1,1}` (UV default), `local_a0..a3 = 0xff` (white fully opaque).
2. If any pointer param is null, replaces with corresponding defaults.
3. Reads viewport from `*(EBX + 0x60)`:
   - `local_90 = float(viewport.width)` from `*(iVar1+0xc)`.
   - `local_8c = float(viewport.height)` from `*(iVar1+0x10)`.
4. `FUN_004c75e0(iVar1, local_98, local_9c)` — reads scissor or clipped region into shorts.
5. `local_88 = float(short) local_98[0]`, `local_84 = float(short) local_9c[0]` — integer-to-float rect offsets.
6. `local_64 = DAT_005cc320 / *(EBX + 0x80)` — Z value for quad vertices (W reciprocal? `[UNCERTAIN]`).
7. Computes 4 screen-space vertex positions:
   - `local_80[4] = viewport.width × param_1[0] + local_88`
   - `local_6c = viewport.height × param_1[1] + local_84`
   - `local_54 = viewport.width × param_2[0] + local_88`
   - `local_50 = viewport.height × param_2[1] + local_84`
   - `local_38 = viewport.width × param_2[0] + local_88` (re-read)
   - `local_34 = local_50`
   - `local_1c = local_38`, `local_18 = local_6c`
8. All 4 vertices share `Z = local_64`, `W = 1.0f` (`0x3f800000`).
9. Writes Z from `*(DAT_007d3ff8 + 0x18)` into vertex Z slots.
10. Copies colour `in_EAX` as packed RGBA into `local_60`, replicated to `local_44`, `local_28`, `local_c`.
11. Render-state setup via vtable `DAT_007d3ff8`:
    - `(DAT_007d3ff8+0x20)(2, 3)` — `SetRenderState(2, 3)`.
    - `(DAT_007d3ff8+0x20)(7, 1)` — `SetRenderState(7, 1)`.
    - `(DAT_007d3ff8+0x20)(1, param_3)` — `SetRenderState(1, param_3)`.
    - If `in_EAX[3] != 0xff`: `(DAT_007d3ff8+0x20)(0xc, 1)` — enable alpha blend.
12. `(DAT_007d3ff8+0x2c)(vertices, 4, 0, 1, 2)` — draw first triangle (V0,V1,V2).
13. `(DAT_007d3ff8+0x2c)(vertices, 4, 2, 3, 0)` — draw second triangle (V2,V3,V0).
14. If alpha blend was enabled: `(0xc, 0)` — disable.
15. `(DAT_007d3ff8+0x20)(7, 2)` — restore render state 7.
