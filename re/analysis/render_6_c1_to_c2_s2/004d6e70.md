param_2 = D3D stage index.

  (D3D vtable DAT_007d4110 offset 0x104).

**Non-null path:**

   cached per-stage in DAT_007d6b34[param_2*6]; emits SetSamplerState(stage,1,v).
2. Filter mode from param_1[0x14] bits 12..15 (mag filter index → same table), cached in
   DAT_007d6b38[param_2*6]; emits SetSamplerState(stage,2,v).
3. If DAT_007d6c10 > 0: reads *(byte*)(DAT_007d6c10 + (int)param_1) as mip-aniso level;
   cached in DAT_007d6b44[param_2*6]; emits SetSamplerState(stage,10,v); if level>1 adjusts
   filter index +7.
4. Filter byte from param_1[0x14] bits 0..7 (possibly adjusted), indexed into 8-byte table
   at 0x005d8ca8; emits SetSamplerState(stage,6,min) + SetSamplerState(stage,5,min) and
   SetSamplerState(stage,7,mag); cached in DAT_007d6b3c[param_2*6].
5. Calls FUN_004d6ce0(*param_1, param_2) to bind the actual texture object for this stage.
