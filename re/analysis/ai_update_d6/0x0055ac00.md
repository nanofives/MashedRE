## Mechanical description

Sets or clears a bit in a physics contact/shape bitset.

**Params:**
- `param_1` = physics world/system object.
- `param_2` = shape/contact descriptor; `*(param_2+0x20)` = shape index (ushort).
- `param_3` = 1 → set bit, 0 → clear bit.

Bit position: `shape_index & 0x1f` within word `shape_index >> 5` of the bitset at `*(param_1+0x5c)`.

This is a simpler variant of `FUN_0055ac50` that only toggles the contact/shape activation flag without the joint-deactivation callback.
