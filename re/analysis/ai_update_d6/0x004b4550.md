## Mechanical description

Algorithm:
1. Initialize accumulator with first point.
2. Sum remaining `param_3 - 1` points into the accumulator (unrolled 4-per-iteration for performance).
3. Multiply by `_DAT_005cc320 / param_3` = `1.0f / param_3`.

Result stored in `param_1[0..2]`.

Called from `FUN_0044e0a0` (type-10 physics impulse handler) to compute the centroid of the impulse vertex set before applying the impulse. The vertex count is at `*(param_1+0x44)` in the caller.
