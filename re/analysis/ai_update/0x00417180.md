## Mechanical description

- Takes `param_1` (int): vehicle index (0-based).
- Manages per-vehicle AI line-bank selection and spline index cycling.

### Switch-request processing (if `DAT_0089a500[param_1 * 0x74]` != 0)

- Reads current AI line type from `(&DAT_0089a4cc)[param_1 * 0x1d]`.
- Writes 1 to `(&DAT_0089a504)[param_1 * 0x74]` (starts switch timer).
- Line-type dispatch:
  - Type 2 (slow lines): increments `DAT_0089a4d0[param_1 * 0x1d]` (spline index); if new index > 2, wraps to 0; calls `FUN_00414030(0xffffffff)`.
  - Type 1 (inside lines): increments spline index; if new index > 2 OR `DAT_008022ac[new_idx * 0x204]` (count field) < 4 → sets line type to 2, spline index to 0.
  - Type 0 (race lines): increments spline index; if new index > 2 OR `DAT_00801ca0[new_idx * 0x81 * 4]` (= new_idx * 0x204, count field) < 4 → sets line type to 1, spline index to 0; calls `FUN_00414030(0xffffffff)`.
  - Type 3 (cheat lines): increments spline index; if new index > 2 OR `DAT_00802ec4[new_idx * 0x204]` < 4 → sets line type to 0, spline index to 0; calls `FUN_00414030(0xffffffff)`.
- Clears switch-request flag: `DAT_0089a500[param_1 * 0x74] = 0`.

### Switch timer

- Reads `iVar1 = DAT_0089a504[param_1 * 0x74]`.
- If iVar1 != 0 AND iVar1 < 9000: increments by `DAT_007f1008` and returns (timer still running).
- Else: clears timer to 0; falls through to line-type reset.

### Line-type reset (after timer expires or timer == 0 at entry)

- If `DAT_0089a368 == 0`: sets line type to 0 (race lines).
- If `DAT_0089a368 == 2`: sets line type to 0 (race lines).
- If `DAT_0089a368 == 1`: if current type != 2, calls `FUN_00414030(0xffffffff)`; sets line type to 2 (slow lines).

### Random spline variation (low-frequency)

- Computes `iVar3 = DAT_007f0ff8 / 3000`.
- If low nibble of `iVar3` == 0xf AND `DAT_0063bd90` != iVar3 (once per period):
  - Calls `FUN_00472650(0, 0x3f800000)` → float10 result; 0x3f800000 = float 1.0.
  - If result < `_DAT_005cc32c` → increments spline index + calls `FUN_00414030(0xffffffff)`.
  - Sets `DAT_0063bd90 = iVar3`.
- Else: clears `DAT_0063bd90 = 0`.
