## Algorithm note

This generates a direction vector on the unit sphere via z-uniform + half-azimuth sampling.  
Azimuth range is `[0, π]` not `[0, 2π]`, so the distribution covers only half the azimuthal circle — output is **not** uniformly distributed over the full sphere. This appears intentional for the caller's use case.
