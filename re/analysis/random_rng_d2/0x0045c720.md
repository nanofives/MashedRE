## Body
`0x0045c720`–`0x0045c7c8`

## Mechanical description

Takes `param_1` = player index (0–3); `param_2` = track index within route.

- Reads `(&DAT_007f1a1c)[param_1 * 4]` — player route type (0–5)
- Switch selects one of 6 music-track table bases:
  - `0`: `DAT_005fd190`
  - `1`: `DAT_005fe268`
  - `2`: `DAT_005ff340`
  - `3`: `DAT_00600418`
  - `4`: `DAT_006014f0`
  - `5`: `DAT_006025c8`
  - default: `0` (null)
- Writes `DAT_0088f680[param_1] = table_base + param_2 * 0x2c`
