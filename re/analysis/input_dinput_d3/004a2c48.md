## Correction of d2 stub description

S-2285 in input_dinput_d2 described this as "button state byte reader." That description reflects its *call context* (called in analog paths of FUN_00497310 and FUN_00496530), not the function's behavior. The function is a general **FPU float80→int64 rounding utility** (ROUND/banker's rounding of x87 ST0 to ulonglong). It is not specific to button state.
