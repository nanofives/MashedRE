- `fpatan(y/x, 1)` = `atan(y/x)` (x87 FPATAN: ST0=y/x pushed, pops → atan)

- Quadrant correction: if `param_2 > 0` (dz positive), add `_DAT_005cd09c`
- Wrap: if result ≥ `_DAT_005ccac4`, subtract full-circle value
- Special: dx=0, dz=0 → return 0.0
