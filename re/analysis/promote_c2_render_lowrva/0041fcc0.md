## Mechanical description

Per-player tire-mark decal renderer. No outer guard (always runs when called).

Body:

1. RGBA placeholder: `local_68 = 0; local_67 = 0; local_66 = 0; local_65 = 0xff;` (black, alpha later overwritten).
2. `local_64 = 0x3f800000; local_60 = local_5c = 0;` (vector 1,0,0).
3. Color-table index: `iVar1 = (*(int*)(*(EDI + 0x268) + 0x80) / 6) * 0xc` — divide a per-vehicle-id integer by 6, multiply by 12 (= sizeof(RGB triple in float)).
4. Color-table A (vehicle body color): copy 3 floats from `&DAT_005f5fe8[iVar1]` into `local_4c..local_44`.
5. Color-table B (skid color): copy 3 floats from `&DAT_005f6088[iVar1]` into `local_58..local_50`.
6. `FUN_004c52f0(local_40, *(*(EDI + 0x26c) + 4) + 0x10, 0)` — copy wheel-transform matrix (frame+0x10 = modelling matrix).
7. `FUN_004c4d20(local_40, &local_64, 0x42b40000, 1)` — rotate-by-axis-angle: axis=(1,0,0), angle 90.0f, flag 1.
8. `FUN_004c51a0(local_40, &local_58, 1)` — RW matrix translate by skid-color-vector (reusing local_58..50 as a 3-float buffer — note: the same local is the second-color table; this is a debug/reuse pattern, but Ghidra type-aliasing).
9. `FUN_004c5010(local_40, &local_4c, 1)` — RW matrix scale by body-color-vector.
10. `FUN_004c45f0(local_40)` — finalise matrix.
11. `local_65 = FUN_004a2c48()` — query alpha (byte).
12. `FUN_004769f0(&local_68)` — submit color.
13. `FUN_00476a10(local_40)` — submit matrix.
14. `FUN_00476d00(DAT_0063e4a0)` — finalise draw with tire-mark texture handle.

Returns.

## Uncertainties / Stubs

- Callees C1+: 0x004c52f0, 0x004c4d20, 0x004c51a0, 0x004c5010, 0x004c45f0, 0x004769f0, 0x00476a10, 0x00476d00, 0x004a2c48.
