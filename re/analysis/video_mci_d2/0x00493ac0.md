## LAB_00493b40 (NT5+ path) — 0x00493b40

Two-instruction stub:
```asm
00493b40:  MOV EAX, 0x3
00493b45:  RET
```
Returns the value `3` in EAX. `CP_THREAD_ACP = 3` is the Windows code-page constant meaning "use the current thread's ANSI code page". Win2000 and later support this sentinel directly in API calls (e.g. `MultiByteToWideChar`, `WideCharToMultiByte`).
