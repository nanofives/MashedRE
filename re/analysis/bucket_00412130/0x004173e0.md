## Mechanical description

- Adds bias +0x200 (= 512) to convert to 0..1024.
- Validates `0 <= biased < 0x401` (=1025) for both axes.
- Coarse cell `(x>>3)*0x80 + (y>>3)` indexes `&DAT_007f1a9c` (2-byte entries, range [1..0x1ff] = 511 cells max).

- On out-of-range or empty cell: returns `0xff` (no-decal sentinel).
