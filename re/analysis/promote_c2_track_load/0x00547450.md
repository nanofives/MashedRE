## Purpose

Sphere vs triangle closest-point overlap test. Returns 1 if the sphere overlaps the
triangle, 0 if separated. Writes the triangle normal and overlap depth to output
parameters. Four phases: axis-aligned slab rejection, vertex proximity, edge proximity,
triangle-plane test.

## Shape

**Signature:** `undefined4 FUN_00547450(float *param_1, float *param_2, float *param_3, float *param_4, float *param_5, float *param_6)`

- `param_1` — sphere: `[center_x, center_y, center_z, r_or_r2]` (see U-3569).
- `param_2` — triangle vertex V0 (3 floats).
- `param_3` — triangle vertex V1 (3 floats).
- `param_4` — triangle vertex V2 (3 floats).
- `param_5` — **output**: triangle normal (3 floats; written by this function).
- `param_6` — **output**: contact depth scalar (1 float; = `-d` where d = signed plane distance).

**Returns:** `undefined4` — 1 = overlap, 0 = separated.

**Phase 1 — axis-aligned slab rejection (3 axes):**
For each axis (X, Y, Z), computes vertex-to-center signed distances. If all three vertices
lie outside the sphere's radius on the same side, returns 0. Uses sign-bit extraction
(`(int)f >> 0x1f`) to count negative projections.

**Phase 2 — vertex proximity (3 vertices):**
Computes squared distances from sphere center to V0, V1, V2. If any distance² < param_1[3],
returns 1 immediately.

**Phase 3 — edge proximity (3 edges: V0→V1, V1→V2, V0→V2):**
Projects sphere center onto each edge's infinite line; if closest-point distance² >
param_1[3] and < a second discriminant, returns 0 on that axis.

**Phase 4 — triangle-plane test:**
Cross product `(V1-V0) × (V2-V0)` → unnormalized normal; normalized via `FastInvSqrt`
(0x004c3b90). Writes normalized normal to `*param_5`. Computes signed distance d =
dot(normal, sphere_center - V0). If |d| > param_1[3], returns 0. Else writes `*param_6 = -d`
and returns 1.
