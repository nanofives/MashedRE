## Purpose

AABB vs triangle Separating Axis Theorem (SAT) test. Returns 1 if the AABB overlaps the
triangle, 0 if separated. Leaf function (no calls). Implements the 12-axis SAT:
3 AABB face axes (phase 1 outcode) + 9 edge-edge cross-product axes (phase 2).

- `param_2` — triangle vertex V0 (3 floats).

**Returns:** `undefined4` — 1 = overlap, 0 = separated.

**Phase 1 — per-vertex outcode (3 face axes):**

Each vertex receives a 6-bit outcode against the AABB planes:

| Bit | Mask | Meaning |
|---|---|---|
| 0 | 0x01 | vertex.x < AABB.min_x |
| 1 | 0x02 | vertex.x > AABB.max_x |
| 2 | 0x04 | vertex.y < AABB.min_y |
| 3 | 0x08 | vertex.y > AABB.max_y |
| 4 | 0x10 | vertex.z < AABB.min_z |
| 5 | 0x20 | vertex.z > AABB.max_z |

- V0 inside AABB (outcode == 0) → return 1 immediately.
- All three vertices in same outside half-space (`outcode_V0 & outcode_V1 & outcode_V2 != 0`) → return 0.

**Phase 2 — 9 edge-edge cross-product axes:**

For each triangle edge (V0→V1, V0→V2, V1→V2) crossed against each AABB face normal
(X, Y, Z), a separating axis inequality is tested. Branch pivot on `DAT_005d757c`
(0.0f, at address 0x005d757c) determines edge direction sign.

Total: 12 SAT axes tested (3 face + 9 edge-edge).
