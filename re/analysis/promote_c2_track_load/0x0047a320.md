## Purpose

BSP_AssetTable, and writes the corresponding resolved DFF handle into a parallel dword
handle array at BSP struct A + 0x2ac0. Both the filename slot and handle array slot are
indexed by the same counter at `DAT_006bf1cc + 0x218` (BSP struct B +0x218).
