Sets the "last RWS error" record at a global error-state slot. Only commits if the slot is empty (avoids overwriting an earlier, more specific error). The sentinel 0x80000000 in the high dword distinguishes "no error" from any real error code. Matches RenderWare's `RwError`/`RwGetLastError` pattern.
