# 0x004648b0 — FUN_004648b0

**RVA:** 0x004648b0

**Signature:** `void FUN_004648b0(void)`  
**Subsystem:** audio  
**Status:** C1  
**Session:** timer_d3_cont1_b

## Mechanical description

Processes a circular ring buffer of audio/MIDI events.

Ring buffer state:
- `DAT_0x007e9de0` — write index (current tail)
- Starting read index: `DAT_007e9de0 - 1` (wraps at 0 → 0xFF, max 256 entries)
- `DAT_0x007f1030` — current frame/tick key to match
- Max iterations: 256

For each ring slot `iVar7` (backwards from write pointer):
- If `DAT_007ea1e4[iVar7] != DAT_007f1030`: sets `bVar2 = true` (stop flag) → breaks
- Reads event type from `DAT_007e9de4[iVar7]`
- Reads event ID from `DAT_007ea5e4[iVar7] - 0xFF00`
- Dispatches to `FUN_00464670(eventId, type, extra, value)` (C1, audio):

| Type (DAT_007e9de4) | Dispatch |
|---------------------|----------|
| 3 | `FUN_00464670(id, 0, 0, 0)` |
| 4 | `FUN_00464670(id, 1, DAT_007ea9e8[iVar7*8], 0)` |
| 8, sub-case 0 | `FUN_00464670(id, 3, 0, DAT_007ec9e8[iVar7*0x10])` |
| 8, sub-case 1 | `FUN_00464670(id, 2, 0, value)` |
| 8, sub-case 3 | `FUN_00464670(id, 3, 0, value)` |
| 8, sub-case 4 | `FUN_00464670(id, 4, 0, value)` |
| 0xB | if `FUN_004a2c48()` == 0 or 1: `FUN_00464670(id, 3, 0, 0)` |

## Notes

- Ring buffer is 256 entries (index wraps 0↔0xFF).
- Iterates backwards until key mismatch or 256 iterations exhausted.
- Event IDs are stored as `raw - 0xFF00`; base range 0xFF00+.
- Type 0xB dispatches based on `FUN_004a2c48()` result (frontend state query).
