# 0x00476d00 — ParticleBatch_AppendEntry
**Confidence:** C2  
**Subsystem:** render  
**Status:** mapped

## Mechanical description

Appends one particle entry to a batch pool struct pointed to by param_1.

Guard: `param_1[0xc] < param_1[0x8]` (count < capacity). If at capacity, no-op.

If space available, copies current emitter state globals into each non-null channel array at index `param_1[0xc]`, then increments `param_1[0xc]`.

### Batch pool struct layout (param_1 offsets, all u32 / ptr)
| Offset | Field | Usage |
|--------|-------|-------|
| +0x08 | capacity | max entries |
| +0x0c | count | current entry count; incremented |
| +0x10 | pos_array | XYZ float[n], stride 0x0c → copies DAT_00692528/2c/30 |
| +0x14 | vel_array | XY float[n], stride 0x08 → copies DAT_006924dc/e0 |
| +0x18 | colour_array | u32[n] → copies DAT_00692554 |
| +0x1c | matrix_array | 4×4 float[n], stride 0x40 (16 dwords) → copies DAT_006924e8..+0x3c |
| +0x20 | scalar_array | u32[n] → copies _DAT_006924d8 |
| +0x24 | rgba_array | 4×u32[n], stride 0x10 → copies DAT_00692598/9c/a0/a4 |
| +0x28 | block_array | 8-dword[n], stride 0x20 → copies DAT_00692534..+0x1c |

### Emitter state globals consumed
`DAT_00692528/2c/30` (pos XYZ), `DAT_006924dc/e0` (vel XY), `DAT_00692554` (colour),
`DAT_006924e8..+0x3c` (matrix 64 bytes), `_DAT_006924d8` (scalar), `DAT_00692598/9c/a0/a4` (RGBA), `DAT_00692534..+0x1c` (block 32 bytes).
