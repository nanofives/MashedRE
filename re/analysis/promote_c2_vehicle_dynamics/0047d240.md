## Purpose

Position a vehicle physics body at given XYZ coordinates and re-integrate into the physics
scene. Looks up the physics body from `DAT_006c9a78[param_1]` (collision body handle),
optionally deregisters from parent, sets transform node (+0x30=X, +0x34=Y, +0x38=Z),
marks dirty, disables body, then re-registers contact. Called with Y=−10.0f to spawn
vehicles below the playfield.

void FUN_0047d240(int param_1, int param_2, float param_3, float param_4, float param_5);
// param_1 = vehicle slot index (0..3)
// param_2 = orient mode (0 = no rotation)
// param_3 = X spawn position
// param_4 = Y spawn position (−10.0f on init)
// param_5 = Z spread offset
