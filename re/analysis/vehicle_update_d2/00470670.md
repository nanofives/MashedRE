### Control zero-clearing

Zeroes per-vehicle control output slots at fixed offsets from `in_EAX`:
- `+0xb20`, `+0xb1c`, `+0xb18`, `+0xb14` — control flags
- `+0x3f4`, `+0x330`, `+0x26c`, `+0x1a8` — force/torque accumulators

Also zeroes `in_EAX + 0xb0c` when `*(float*)(in_EAX + 0x9e4) == 0.0` (zero-speed case).

### Speed-normalized forward-projection

When speed != 0:
- Reads velocity: `in_EAX + 0x9b0` (X), `+0x9b4` (Y), `+0x9b8` (Z)
- Reads forward vector: `in_EAX + 0x9d4` (X), `+0x9d8` (Y), `+0x9dc` (Z)
- Computes dot product of forward × velocity; takes abs value
- Writes to `in_EAX + 0xb0c`: `(dt - |dot| / speed) * speed` where `dt = _DAT_005cc320`

4. Multiplies by `_DAT_005cc32c` (delta-time scale).
