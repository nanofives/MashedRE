## Mechanistic description

Single-parameter function (`param_1` = tick delta, uint). Performs one complete physics
tick for the in-race simulation:

1. Calls `FUN_00423b00()` — pre-tick state update (0x00423b00).
2. Guards on `DAT_007f1a50 == 1` (pause flag); if paused and conditions not met (`DAT_007f1a54 != 4` or both `DAT_007f1a74 == 0 && DAT_007f1a70 == 0`), returns 0 immediately.
3. Increments `DAT_007f1a78` when pause-override is active.
4. Calls `FUN_00418860()` — input/control state update (0x00418860).
5. Calls `FUN_00424eb0(param_1, &DAT_00803324)` — AI update (0x00424eb0).
6. Calls `FUN_00470c70(param_1, &DAT_00803324)` — vehicle physics update dispatcher (0x00470c70).
7. Calls `FUN_00422ba0(param_1)` — collision event processor (0x00422ba0).
8. Calls `FUN_004252c0(param_1)` — post-collision/item step (0x004252c0).
9. Returns 1 on success.
