## Mechanistic description

No-parameter function. Processes a circular event queue (`DAT_007e9de4/ea1e4/ea5e4`,
max 0x100 entries). Reads up to 0x100 events backwards from `DAT_007e9de0 - 1`,
stopping when queue entry's session-ID `(&DAT_007ea1e4)[idx]` doesn't match
`DAT_007f1030` (current session token) or when `local_80 = 1` (done flag).

Snaps current vehicle states via `FUN_0046cbb0(i, &state_out, &input_out)` for
vehicles 0..3 at function entry.

### Event type dispatch (field `(&DAT_007e9de4)[idx]`)

**Type 10 — position reset:**

- Reads position `DAT_007eca14/18/1c + idx*0x40`.
- Calls `FUN_0044e0a0(local_58)` (0x0044e0a0) — applies matrix/position to vehicle.

**Type 9 — [UNCERTAIN]:**

**Type 8 — [UNCERTAIN]:**
- Reads impact position from `DAT_007ec9e8 + idx*0x10 * _DAT_005cd0ec`.
- Calls `FUN_0047cea0(vehicle_id, &pos)` (0x0047cea0) — likely hit/explosion effect.

**Type 6 — vehicle state change:**
- Reads vehicle color from `(&DAT_007ea9e4)[idx*8]`.
- If color == -0xffff01 or -0xff80 (team colors A): `FUN_0046cbe0(vehicle, 2, 0)` + `FUN_004922e0(vehicle, 2, 0x1e, 0xff)`.
- If color == -0xfefeff (team color B): `FUN_0046cbe0(vehicle, 1, 0)`.

**Type 4 — vehicle-vehicle impact:**
- Reads vehicles A=`(&DAT_007ea9e4)[idx*8]`, B=`(&DAT_007ea9e8)[idx*8]`.
- Calls `FUN_004922e0` for both vehicles.
- If both vehicles alive (state == 0):
  - Updates collision pair table `DAT_008995cc/dc/ec/f0/f4`:
    - `DAT_008995ec[A*0x4e] = B + 1000`, `DAT_008995ec[B*0x4e] = A + 1000`
    - `DAT_008995f0[A*0x4e]` and `[B*0x4e]` = current game-time `DAT_007f0ff8`
    - `DAT_008995cc[A*0x4e + B]` and `[B*0x4e + A]` = game-time
    - `DAT_008995dc[A*0x4e + B]` = `DAT_008995dc[B*0x4e + A]` = impact speed magnitude (FUN_004c3ac0)
    - `DAT_008995f4[A*0x4e]`, `[B*0x4e]` = [0 or 1 based on impact data]
  - Calls `FUN_00417370(A)` and `FUN_00417370(B)` if not AI (`FUN_0040e470 != 1`).

**Type 3 — vehicle collision with speed check:**
- Increments `DAT_008995b0[vehicle * 0x4e]` (collision counter).
- Reads impact velocity from `DAT_007ec9f4/f8/fc + idx*0x40`.
- Computes speed magnitude via `FUN_004c3ac0`.
- If magnitude > `_DAT_005cc568`: calls `FUN_004922e0(vehicle, 1, 2, 0xff)`.
- If magnitude > `_DAT_005cd098`: calls `FUN_00480720(vehicle)` (speed correction/respawn) + `FUN_00480d60(vehicle, 7)`.
