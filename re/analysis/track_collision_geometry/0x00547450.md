- `iVar23 == 0` (all positive) AND all projections ≥ `param_1[3]` → separated (return 0).
- `iVar23 == -3` (all negative) AND all `(param_1[3] + proj)` have the top bit set → separated (return 0).

**Phase 2 — vertex proximity tests.**

Computes squared distances from sphere center to each vertex (`fVar2`, `fVar3`, `fVar4`). If any distance² < `param_1[3]` (interpreted as radius or radius²), returns 1 immediately.

**Phase 3 — edge proximity tests (3 edges).**

For each edge (V0→V1, V1→V2, V0→V2):
1. Projects the sphere center onto the infinite line through the edge using dot-product / length².
2. Computes the closest point on the line.
3. Computes distance² from sphere center to that closest point.
4. If distance² > `param_1[3]` AND distance² < second discriminant → returns 0 (separated).

**Phase 4 — triangle-plane test and normal output.**

Computes signed distance from sphere center to triangle plane:
`d = dot(normal, sphere_center - V0)`.

If `d < -param_1[3]` or `d > param_1[3]` → return 0 (sphere does not straddle the triangle plane).

Writes `*param_6 = -d` (contact depth along normal).

Returns 1 (overlap) or 0 (separated) depending on all tests.
