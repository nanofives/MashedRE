| 0x19 (X1R5G5B5) | LAB_004debb0 |
| 0x1a (R5G6B5) | LAB_004debf0 |
| 0x1b (A4R4G4B4) | LAB_004dec70 |

Then dispatches on `param_2[5]` (bpp):
- **4bpp / 8bpp**: iterates rows, calls `(*pcVar3)(dst_pixel_ptr, src_byte)` per pixel.
- **16bpp**: iterates rows, calls `(*pcVar3)(dst_pixel_ptr, src_word)` per pixel.
- **32bpp**: iterates rows, calls `(*pcVar3)(dst_pixel_ptr, src_dword)` per pixel.
- **default**: logs error `0x8000000c` via `FUN_004d7ff0` + `FUN_004d8480`.
