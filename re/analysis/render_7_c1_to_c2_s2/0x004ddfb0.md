## Mechanical description

Inverse of `FUN_004ddd60` (0x004ddd60). If `param_3 & 0x2000`: direct 32-bit copy (`*param_1 = *param_2`) and returns 1.

Otherwise dispatches on `param_3 & 0xf00`:

| nibble | D3D format | Expansion |
|--------|-----------|-----------|
| 0x100 | A1R5G5B5 | A = bit15→0xFF/0x00; B = bits4..0 << 3; G = bits9..5 << 3; R = bits14..10 << 3 |
| 0x200 | R5G6B5 | A=0xFF; B = bits4..0 << 3; G = bits10..5 << 2; R = bits15..11 << 3 |
| 0x300 | A4R4G4B4 | A = nibble15..12 (0xFF if = 0xF); R = nibble11..8; G = nibble7..4; B = nibble3..0 |
| 0x400 | L8 | A=0xFF; RGB all = byte(src>>8) |
| 0x500 | A8R8G8B8 | direct byte shuffle: B=byte0, G=byte1, R=byte2, A=byte3 |
| 0x600 | X8R8G8B8 | A=0xFF; B=byte0, G=byte1, R=byte2 |
| 0xa00 | X1R5G5B5 | A=0xFF; B = bits4..0 << 3; G = bits9..5 << 3; R = bits14..10 << 3 |

On unrecognized format: calls `FUN_004d7ff0(0x8000000d)` + `FUN_004d8480`; returns 0.
