## Mechanical description

1. Checks `*(byte*)(DAT_00911ae4 + 10 + param_2) & 0xf0`: if non-zero (compressed raster), logs error "Conversion_from_compressed_raste..." via `FUN_004d7ff0(2, ...)` and returns 0.

2. Locks source raster if needed (`FUN_004c7860(param_2, 0, 2)` if bit 1 of `*(param_2+0x22)` clear); sets a restore flag.

3. Checks cubemap/mipmap flags at `*(param_2+0x23) & 0x60` — if set and not already native, calls `FUN_004c76c0(param_2, 2)` to convert.

4. Dispatches on destination bit-depth `*(param_1+0xc)`:
   - **0x20 (32bpp):** calls `FUN_004de390(param_1, param_2)` — full format conversion.
   - **0x08 (8bpp palettized):** if source is also 8bpp palettized and dims match: bulk memcpy (word-aligned) from `param_2+4` to `param_1+0x14`; also copies or generates the 256-entry palette at `param_1+0x18`. If dims differ: row-by-row copy. On mismatch: logs "Conversion_from_non_palettised_r..." error.
   - **other:** logs "Conversion_from_rasters_to_4bit_i..." error.

5. On success: unlocks raster and native context if step 2/3 locked them; returns 1.
