# 0x004df120 — FUN_004df120 (RW raster ← RW image pixel converter, with palette/format selection)

**RVA:** 0x004df120  
**Size:** 0x004df120..0x004df6ab (0x58c = 1420 bytes)  
**Subsystem:** render  
**Confidence:** C2  
**Batch:** batch-render-7-s2  
**U-id:** U-5535

## Signature

```c
undefined4 FUN_004df120(int *param_1, int param_2);
```

- `param_1` — destination: RW raster object.
- `param_2` — source: RW image object.

## Mechanical description

1. Checks `*(byte*)(DAT_00911ae4 + 10 + raster) & 0xf0` (compressed raster): logs error "Conversion_to_compressed_rasters..." and returns 0 if set.

2. Determines destination format:
   - If `*(char*)(raster+8) == 0`: calls `FUN_004dee40(image)` to auto-select format, stores result byte at `raster + DAT_00911ae4 + 8`.
   - Otherwise: uses `*(int*)(raster + DAT_00911ae4 + 0x18)` (existing D3DFORMAT), with special handling for 0x29 (D3DFMT_A8P8: forced 0x01 if src format nibble == 5, else 0).

3. Propagates format to mipmaps if `raster != *raster`.

4. Rescales source image if dims differ (`FUN_004db2e0`).

5. Locks raster (`FUN_004c7860(raster, 0, 5)`) unless already native — bVar7 flag.

6. Optionally locks native format if mipmapped/cubemapped (`FUN_004c76c0`).

7. Dispatches on `raster[5]` (destination bpp):
   - **4bpp / 8bpp**: calls `FUN_004dfbc0(param_1, rescaled_img)` — indexed color pack.
   - **16bpp**: if source is 4/8bpp palettised: uses `FUN_004df750(D3DFMT_id)` to get encoder, converts palette entries to 16bpp, then maps indexed pixels. If source is 32bpp: switches on `D3DFMT_id` for hardcoded 32→16bpp paths (0x17 R8G8B8, 0x18 A1R5G5B5, 0x19 X1R5G5B5, 0x1a R5G6B5, 0x1a A4R4G4B4) or uses `FUN_004df750` for the generic case.
   - **24bpp / 32bpp**: calls `FUN_004e02d0(param_1, rescaled_img)`.
   - **0x40 / 0x80**: no-op.

8. Unlocks native format and raster if locked in step 6/5.

9. Frees rescaled image if created in step 4.

10. Returns 1.
