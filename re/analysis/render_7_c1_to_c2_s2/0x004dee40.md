# 0x004dee40 — FUN_004dee40 (D3D9 best-format selector for RW raster)

## Signature

```c
int FUN_004dee40(int param_1);
```

- `param_1` — RW raster image object handle.
- **returns** — a format selector integer (upper nibbles = format ID, used as `param_3` for `FUN_004ddd60` / `FUN_004ddfb0`).

## Mechanical description

Reads bit-depth at `*(param_1+0xc)`:

**4bpp or 8bpp path:**
Scans the palette at `*(param_1+0x18)` checking alpha byte (offset +3 of each 4-byte entry):
- If `alpha != 0xFF` and `alpha > 0x0F`: returns 0x4500 or 0x2600/0x4600 depending on format.
- Uses `FUN_004d38b0(0x15)` to check if A8R8G8B8 (0x15) is supported → returns 0x500 (A8R8G8B8).
- Else checks `FUN_004d38b0(0x1a)` → returns 0x100 + 0x200 (A1R5G5B5 or R5G6B5).
- If all palette entries fully opaque: returns 0x4600 or 0x2600 (palettised format selectors).

**16bpp / 32bpp path:**
Scans the pixel buffer at `*(param_1+0x14)`:
- Scans row-by-row, checks alpha byte `*(row_ptr + 3)`:
  - `alpha == 0xFF`: fully opaque — continue.
  - `alpha <= 0x0F`: partially transparent → set flag.
  - else: fully transparent / > 0x0F boundary → `goto LAB_004deede`.
- After full scan: if flag set returns 0x100 (A1R5G5B5) or 0x400 (A8R8G8B8) based on caps check.
- If all opaque: returns 0x200 (R5G6B5) or 0x400.

Returns a packed format code used as `param_3 & 0xf00` nibble by callers.
