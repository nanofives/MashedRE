## Mechanical description

Dispatches on `param_3 & 0xf00` (format nibble):

| nibble | D3D format | Output encoding |
|--------|-----------|-----------------|
| 0x000 | default | recursive call with 0x500 then copy |
| 0x100 | A1R5G5B5 | `(alpha1 << 15) | (R>>3 << 10) | (G>>3 << 5) | (B>>3)` |
| 0x200 | R5G6B5 | `(R>>3 << 11) | (G>>2 << 5) | (B>>3)` |
| 0x300 | A4R4G4B4 | `(A>>4 << 12) | (R>>4 << 8) | (G>>4 << 4) | (B>>4)` |
| 0x400 | L8 (luminance) | `((R*0x1e + G*0x3b + B*0xb) / 100) * A / 0xff` |
| 0x500 | A8R8G8B8 | `CONCAT(A, R, G, B)` — direct 32-bit pack |
| 0x600 | X8R8G8B8 | `0xff000000 | (R<<16) | (G<<8) | B` |
| 0xa00 | X1R5G5B5 | `(R>>3 << 10) | (G>>3 << 5) | (B>>3)` |

On unrecognized format: calls `FUN_004d7ff0(0x8000000d)` (error log) + `FUN_004d8480` (error dispatch); returns 0.
