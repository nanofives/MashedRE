## Mechanical description

If `*param_2 == 0`: writes 0x54 to `*param_1` and returns 1. Otherwise:

1. Selects a base header size based on flags at `*(byte*)(obj+0x23)`:
   - bit 6 (0x40): header = 0xe4 (cubemap variant).
   - bit 5 (0x20): header = 0x464 (large cubemap variant).
   - else: header = 0x64.

2. Reads the D3D9 interface pointer from `(DAT_00911ae4 + obj)` (the global D3D9 device context table).

3. Dispatches on mip-type field `*(byte*)(base+9) & 0xf` and `*(byte*)(base+10) & 0xf`:
   - For non-cubemap with mipmaps: iterates mip levels via vtable calls at `vtable+0x34` (GetLevelCount), `+0x44` (LockRect), `+0x4c` (GetLevelDesc), `+0x50` (UnlockRect); accumulates `4 + stride*height` bytes per level.
   - For cubemap with mipmaps: same per face × 6.

4. Writes accumulated total to `*param_1`; returns 1.
