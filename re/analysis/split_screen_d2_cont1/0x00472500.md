## Interpretation

**PlayerColorGet** — returns one of two color variants for the given player index. Each player has a 0x10-byte (16-byte) entry in the color table starting at `DAT_00691500`:
- Offset +0: alternate color (returned when toggle byte = 0)
- Offset +4: selected/primary color (returned when toggle byte ≠ 0)
- Offset +8: toggle byte at `DAT_00691508 + param_1*4`

Used in D-8444 (PerPlayerGroundTriangleCollect) to build an exclusion list of 5 player-specific colors (indices 0..4). Ground triangles whose RGBA matches any of these are not rendered (they are the player vehicle body colors — skipped to avoid z-fighting with the vehicle mesh).
