## Mechanical description
Sets/clears a "discontinuity" flag on a media-buffer descriptor's bound object, by mode tag
`*(param_1 + 0x10)`. `param_1` = descriptor, `param_2` = flag value (set if non-zero).

- Mode == 1 (IMediaSample-backed): installs a SEH frame (`&DAT_005d0b3c`, handler return 0x5be031),
  then:
  - `hr = (**(*sample))()` — vtable +0 (QueryInterface-shaped to IMediaSample2).
  - If `hr >= 0`: `(**(qi + 0x4c))(...)` — vtable +0x4c (GetProperties-shaped, struct size 0x30).
    If ok: clear flag bit `0x200` when the queried discontinuity dword is 0, else set bit `0x200`;
    then `(**(qi + 0x50))(qi, 0x30, &props)` — vtable +0x50 (SetProperties-shaped).
- Mode == 2 (raw-buffer-backed): `hdr = *(param_1 + 0x0c)`; if `param_2 != 0`: `*hdr |= 8`; else
  `*hdr &= ~8`.
- Returns void.
