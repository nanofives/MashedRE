## Mechanical description
Thread-safe PUSH of a COM interface pointer into the ring buffer. `param_1` = ring object,
`param_2` = COM interface pointer to enqueue. Returns 1 on success, 0 if the ring is full.

1. `WaitForSingleObject(param_1[4], INFINITE)` — acquire the ring semaphore (`+0x10`).
2. Read slot at `param_1[3] + param_1[0]*4` (backing[write-index]). If non-zero (slot occupied → ring
   full): `ReleaseSemaphore(param_1[4],1,NULL)`; return 0.
3. `(**(param_2 + 4))(param_2)` — call vtable slot +4 on the object (IUnknown::AddRef-shaped) to take a
   reference.
4. Store `param_2` into `backing[write-index]`.
5. Advance write index `param_1[0]`; if it equals capacity `param_1[2]`, wrap to 0.
6. `ReleaseSemaphore(param_1[4],1,NULL)`; return 1.
