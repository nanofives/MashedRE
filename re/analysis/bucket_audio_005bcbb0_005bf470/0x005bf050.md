## Mechanical description
Broadcasts a vtable call to every output pin of a CBaseFilter. `param_1` = filter. Returns the last
negative HRESULT encountered, or 0.

For each pin in the array at `+0x11c` (count `+0x120`):
1. `pin = array[i]`; `hr = (**(pin + 0x24))(pin, &dir)` — vtable +0x24 (QueryDirection-shaped),
   writing the direction into a local.
2. If `hr >= 0` and `dir == 1` (output pin):
   - `inner = *(pin + 0x114)` — the pin's inner/connected object.
   - `hr = (**(inner + 0x38))(inner)` — vtable +0x38 (a no-arg state/notify method).
   - If `hr < 0`: remember it as the result.
3. Continue to the next pin.
Return the remembered HRESULT (0 if none failed).
