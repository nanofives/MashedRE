## Mechanical description
Releases a media-buffer descriptor's bound object according to its mode tag `param_1[4]`, then zeroes
the descriptor. `param_1` = descriptor.

- Mode `[4] == 1` (IMediaSample-backed): `(**(*param_1[3] + 8))(param_1[3])` — vtable +8
  (Release-shaped) on the bound sample; `param_1[3] = 0`; fall to common-zero.
- Mode `[4] == 2` (raw-buffer-backed):
  - `cb = *(param_1[3] + 0x10)` — a free-callback record on the bound object.
  - `(*(code *)*cb)(param_1[3], cb[1])` — call the function pointer `cb[0]` with `(object, cb[1])`
    (custom destructor/free).
  - `param_1[3]=0`; zero `[0],[1],[2],[4]`; return.
- Common-zero (mode 1 or mode 0/other): `param_1[0]=0; param_1[1]=0; param_1[2]=0; param_1[4]=0`.
