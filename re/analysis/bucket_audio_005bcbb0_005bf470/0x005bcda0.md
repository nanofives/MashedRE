## Mechanical description
Returns the ring buffer's current occupancy (number of queued items). `param_1` = ring object. Locked
read; releases the semaphore on every return path.

1. `WaitForSingleObject(param_1[4], INFINITE)`.
2. `w = param_1[0]` (write index), `r = param_1[1]` (read index).
3. If `w != r`:
   - if `w < r`: release; return `r - w`.
   - else: release; return `w - r`.
4. Else (`w == r`, ambiguous empty/full): inspect `backing[w]` (`param_1[3] + w*4`):
   - if non-zero (full): release; return capacity `param_1[2]`.
   - else (empty): release; return 0.

Note: step 3 reads the read index from `+0x04` (`[1]`) here, whereas POP advances `+0x04` and PUSH
advances `+0x00`; the difference of the two indices, disambiguated by the slot-occupancy probe at
`w==r`, yields the fill count.
