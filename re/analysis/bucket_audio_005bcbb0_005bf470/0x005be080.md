setter FUN_005bdff0). `param_1` = descriptor. Returns 1 if the flag is set, else 0.

- Mode `*(param_1 + 0x10) == 1` (IMediaSample-backed):

  - If `hr >= 0`: `(**(qi + 0x4c))(qi, 0x30, &props)` (GetProperties-shaped, size 0x30); if ok and
    the property flags have bit `0x200` set → result = 1; then `(**(qi + 8))(qi)` (Release).
- Mode == 2 (raw-buffer-backed): if `(*(byte *)*(param_1 + 0x0c) & 8) != 0` → return 1.
- Else return 0.
