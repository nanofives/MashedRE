## Mechanical description
Binds (loads) a media-buffer descriptor from a source object. `param_1` = descriptor, `param_2` =
source object, `param_3` = mode, `param_4` = explicit length (mode 2). Returns 1 on success, 0 on
fail/empty.

- If `param_2 == 0`: return 0.
- Mode `param_3 == 1` (IMediaSample-backed):
  1. `param_1[3] = param_2` (bind object).
  2. `hr = (**(param_2 + 0xc))(param_2, param_1)` — vtable +0x0c (GetPointer-shaped); writes the data
     pointer into `param_1[0]`.
  3. If `hr >= 0`: `param_1[2] = (**(*param_1[3] + 0x2c))(param_1[3])` — vtable +0x2c
     (GetSize/GetActualDataLength-shaped) → length; set `param_1[4] = 1` (mode); `param_1[1] = 0`
     (pos); return 1.
  4. Else `param_1[3] = 0`; return 0.
- Mode `param_3 == 2` (raw-buffer-backed):
  1. `param_1[3] = param_2`; `param_1[0] = param_2[1]` (data base = source `+0x04`);
     `param_1[2] = param_4` (length).
  2. Falls to common tail.
- Common tail (mode 2 and other non-1 modes): `param_1[4] = param_3` (mode); `param_1[1] = 0`;
  return 1.
