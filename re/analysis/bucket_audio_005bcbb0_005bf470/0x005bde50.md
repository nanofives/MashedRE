## Mechanical description
Zeroes a media-buffer descriptor. `param_1` = descriptor (5-dword struct).

Writes 0 to `param_1[0]`, `param_1[1]`, `param_1[2]`, `param_1[3]`, `param_1[4]` (offsets
+0x00,+0x04,+0x08,+0x0c,+0x10). Returns void.

This clears all five descriptor fields without invoking any bound-object release (contrast
FUN_005bdef0, which releases by mode tag before zeroing). Used by the streaming pumps to drop a
descriptor whose underlying object was already detached.

## Uncertainties
- [UNCERTAIN] U-7114 — media-buffer descriptor layout: `[0]`=data base ptr, `[1]`=current position,

  is consistent across FUN_005bde50/de70/def0/df40 but not independently confirmed by a struct def.
