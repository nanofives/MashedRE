## What it does
CPUID support detection + bit-26 query (SSE2 only).

Identical EFLAGS bit-21 toggle boilerplate as `FUN_004dcf90` / `FUN_004dcff0`.

**If CPUID supported:**
- `cpuid_Version_info(1)` — CPUID leaf 1.
- Returns `result[+8] >> 0x1a & 1` — bit 26 of EDX = SSE2 feature flag.

## Tentative name
`rwD3D9CheckSSE2` (or `cpuid_check_bit26_edx`)

## Relationship
Trio member: `FUN_004dcf90` (bit 23 = MMX), `FUN_004dcff0` (bits 25–26 = SSE/SSE2), this (bit 26 = SSE2 only).
Distinction from `FUN_004dcff0`: returns only the SSE2 bit in isolation vs. the OR of SSE + SSE2.
