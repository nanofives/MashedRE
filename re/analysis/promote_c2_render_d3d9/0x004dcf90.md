## What it does
CPUID support detection + bit-23 query.

**CPUID support check** (EFLAGS bit-21 toggle test):
1. Reads EFLAGS; ORs with 4 (`uVar2`).
2. Flips bit 21 (CPUID ID flag) → `uVar3 = uVar2 ^ 0x200000`.
3. Pushes `uVar3` to EFLAGS, pops back, checks if bit 21 survived.
4. If unchanged → CPUID not supported → return `0`.

**If CPUID supported:**
- `cpuid_Version_info(1)` — CPUID leaf 1.
- Returns `result[+8] >> 0x17 & 1` — bit 23 of EDX = `MMX` feature flag.
