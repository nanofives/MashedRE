## What it does
Computes the magnitude (length) of a 2D vector using RenderWare's fast square-root table.

1. Computes `x² + z²` (squared magnitude).
2. If zero → returns 0.0.
3. Applies the RW fast-sqrt algorithm:
   - `DAT_007d3ff8` = RW device vtable base.
   - `DAT_007d3ffc` = offset into vtable for the sqrt table base pointer.

   - `(sum + 0x800) >> 12 & 0xFFF` = 12-bit table index from the float bit pattern.
   - `table[index] + ((sum + 0x800) >> 1 & 0x3FC00000)` = reconstructed float sqrt.
4. Returns result as `float10` (x87 extended).
