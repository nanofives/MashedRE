## What it does
CPUID support detection + bits-25/26 query (SSE / SSE2 check).

Identical EFLAGS bit-21 toggle boilerplate as `FUN_004dcf90`.

**If CPUID supported:**
- `cpuid_Version_info(1)` — CPUID leaf 1.
- Returns `(result[+8] & 0x6000000) != 0` — tests bits 25 AND 26 together.
  - EDX[25] = SSE.
  - EDX[26] = SSE2.
- Returns `true` if either SSE or SSE2 is present.
