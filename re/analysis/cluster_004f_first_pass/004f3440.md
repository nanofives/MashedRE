## Mechanical description
- Strip-merger / post-processor.
- Takes (int *param_1 [linked list of strip nodes], int param_2 [allow-swap flag]).
- If *param_1 == 0, returns 0.
- Allocates a new strip-list head and computes total size = sum over all strips of (size + 6) (+6 = degenerate-triangle padding).
- Allocates a single contiguous index buffer of total*4 bytes.
- Copies the first strip's indices into the buffer.
- For each remaining strip:
  - Searches for a strip starting/ending with a vertex matching the last vertex in the buffer.
  - If matching first-pair (offsets 0/2): emit single degenerate (or double if parity-mismatch + swap allowed).
  - Else if matching reversed-pair (offsets -4/-2 ago): different degenerate insertion.
  - Else: insert a 2-3 vertex degenerate bridge.
  - Then copy that strip's indices.
- Returns 1.
- Concatenates multiple disjoint strips into ONE long strip via degenerate-triangle bridging — a standard tri-strip optimizer pass.
