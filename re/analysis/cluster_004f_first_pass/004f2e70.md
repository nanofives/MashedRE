## Mechanical description

  - 0x24-byte (36) node free-list (via FUN_004cc7f0 RwFreeListCreate).
  - 0x10-byte (16) edge free-list with capacity param_1/2 + 1.
- For each of param_1 triangles, allocates a node from the first free-list:
  - Sets node[0] = triangle index.
  - Zeroes node+0x10 (parent), node+0x14 (back-ref), node+0x18 (edge-flag), node+0x20 (fanout count).
  - Calls FUN_004f2f90 three times to build edge entries from the triangle's 3 directed edges (puVar4[-2], puVar4[-1], *puVar4 — vertex indices read with 0x14-stride).
  - Stores edge node pointers into node[+4, +8, +0xc] (the 3 neighbors).
- Returns the node-array.
- param_2[0] holds the node free-list handle; param_2[1] holds the edge free-list handle.
