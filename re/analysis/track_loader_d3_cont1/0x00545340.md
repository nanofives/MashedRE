## Role

`FUN_00546380` (0x00546380) with the waypoint set (param_1) and its control-point
array (param_2). Dispatches on type field `*(param_1+0xc)`:

- **Type 1 (open spline):** `degree = *(param_1+0x8) - 2`, clamped to ≤10.
  Reads basis matrix pointer from `DAT_007dc67c[degree]` (0x007dc67c). If non-null,
  iterates `degree` output points; for each, performs a dot-product of the basis
  row against strided (3-float) control-point entries, writing output floats to
  `param_1+0x44+`. Copies first and last control points into `param_1+0x38` and
  the trailing sentinel slot.
- **Type 2 (closed spline):** `degree = *(param_1+0x8) - 3`, clamped to ≤10.
  Builds two knot-coefficient arrays via `DAT_007d3ff8+0x108` (alloc) using a
  Chebyshev recurrence: `coeff[n+2] = coeff[n+1]*4 - coeff[n]`, starting [1,4]
  or [1,2]/[1,5] depending on even/odd degree. Divides by `DAT_005cc320 /
  (float)coeff[half-1]` to normalize. Applies coefficients cyclically (mod
  count) to param_2 control points; stores scaled results at `param_1+0x48+`.
  Frees temp coefficient arrays via `DAT_007d3ff8+0x10c`. Copies first/middle/last
  evaluated points into `param_1+0x38/0x44/0x50` for endpoint anchoring.
- **Other type:** error path — calls `FUN_004d7ff0(1)` then `FUN_004d8480`.

Returns param_1 on success, 0 on alloc failure.
