## Mechanical description
- Pin / stream start (state transition to Running):
  - EnterCriticalSection(param_1 + 0x94).
  - If `param_1[0x19] != 1`:
    - Sets `param_1[0x19] = 1` (state = Running).
    - `timeBeginPeriod(1)` — boost mmtimer resolution to 1ms.
    - Calls vtable `+0x40` (begin / commit).
    - If `param_1[0x1b] == 0`: calls vtable `+0x60` (no-op-handler) and returns.
    - Else: calls vtable `+0x54` with `param_1[0x1b]`. If returns 0, `SetEvent(param_1[0x15])`.
  - LeaveCriticalSection.
