## Mechanical description
- Spawn into the 64-slot "shockwave" particle bank (bitmap at `DAT_007200d0`, stride
  per record 0x18 bytes).
- Linear scan first: walks slots 0..0x40, returns first free slot
  (`(&DAT_007200d0)[i] == 0`). On hit sets the slot to 1 and goes to write path.
- Fallback "find oldest" if no free slot — compares 8 floats at
  `DAT_0071fa50, DAT_0071fa68, DAT_0071fa80, ...0xfaf8` (stride 0x18 bytes), tracks
  max, picks that index.
- Initializes the 0x18-byte record at `0x0071fa40 + slot * 0x18`:
  - +0x14 = `param_2` (color/lifetime arg)
  - +0x10 = 0 (lifetime t)
  - +0x0c = 0 (alpha or phase)
  - +0x00..+0x08 = vec3 from `param_1` (with +0x04 += `_DAT_005cc56c` Y offset).
