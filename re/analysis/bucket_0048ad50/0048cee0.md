## Mechanical description
- Per-type particle bank initializer. Function takes its "type index" via EDI
  (`unaff_EDI`), and uses a switch to pick a texture name:
  - case 1 → "flatshad"
  - case 13, 14, 16 (0xd, 0xe, 0x10) → "wfall"
  - default → "smoke" (fall through after acquiring the texture once)
- Calls `FUN_004770c0(&DAT_00722470 + idx*0x40, 0x817, 0x100, texture)` — sets up
  the per-type immediate buffer (256 quads each, mode 0x817).
- Calls `FUN_00476cb0(&DAT_00722470 + idx*0x40, 5, 6)` — sets alpha-blend mode 5,6.
- Clears `*(&DAT_007668b0 + idx*4) = 0` — per-type write index.
