## Purpose

Collision-win event trigger. Called from VehicleCollisionBroadPhase (0x004709a0) when two vehicles satisfy the proximity collision condition and the recording mode is armed. Iterates 4 channels, fires effect code 3 on each, records winning vehicle index and starts a 6000-unit countdown timer.

## Shape

**Signature:** `void VehicleCollisionWinTrigger(int param_1)` — param_1 = vehicle index
**Returns:** void
**Reads:**
- `(&DAT_00881F90)[param_1 * 0x341]` — player-vehicle flag (must be 0)
- `(&DAT_008815A4)[param_1 * 0x341]` — active contact data (must be non-zero)

**Writes:**
- `DAT_007f0fd4` — winning vehicle index
- `DAT_007f0fdc` — collision countdown timer (set to 6000)
