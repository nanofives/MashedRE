## Purpose

Contact history table scan. Given a geometry entry pointer and a vehicle struct pointer, scans 32-entry contact history at vehicle_struct+0xBFC to find if a live slot's contact ID matches the geometry entry's ID. Returns 1 (found) or 0 (not found).

## Shape

**Signature:** `undefined4 VehicleContactHistoryLookup(int param_1, int param_2)`
- param_1 = geometry entry ptr (field +0x34 = contact ID)
- param_2 = vehicle struct ptr
**Returns:** 1 if contact found in history, 0 if new contact

**Reads (param_2-relative):**
- `+0xBFC..+0xBFC+32*4` — contact history table, 32 int entries
- `piVar1[0x20]` — active flag 0x80 bytes ahead per entry
