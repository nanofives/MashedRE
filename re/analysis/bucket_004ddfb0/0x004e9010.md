## Mechanical description
- Signature `int FUN_004e9010(int geom)`. Geometry-body serialized-size calculator.
- If `(geom[8] & 0x1000000) != 0` (native-only flag): returns 0x28 (header only).
- Else:
  - Base = 0x10 (struct header).
  - If has-prelights (flag bit 8): adds `numVerts*4`.
  - Adds `(numTexSets * numVerts + numTriangles) * 8` (texcoords + triangles).
  - Walks morph-targets at `geom[0x5c]` (each is 0x1c bytes, fields `[5]`=verts, `[6]`=normals, `[1..4]`=center+radius):
    - For each: adds 0x18 (sphere + has-verts + has-normals booleans).
    - If `m[5] != 0` (verts present): adds `numVerts*0xc`.
    - If `m[6] != 0` (normals present): adds `numVerts*0xc`.
