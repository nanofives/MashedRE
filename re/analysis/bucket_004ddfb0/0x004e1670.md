## Mechanical description
- Signature `int FUN_004e1670(int heap, int *slot, int size_aligned)`.
- Splits a free-list block to satisfy an allocation:
  - If remaining space (`*slot - size_aligned`) < 0x100, returns the entire block: removes the slot from the array by swapping with the last entry, decrements heap[0x14], clears the block's free-marker (cell[+0xc] = 0), returns `block_addr + 0x20`.
  - Otherwise: splits the block in two. Updates the original block's size to the requested portion, creates a new free-block header at `block + 0x20 + size_aligned`, links its `prev` to the original block and inherits the original's `next`. Updates the slot entry to point to the new (smaller) free block. Clears the allocated block's free-marker. Returns `block_addr + 0x20`.
