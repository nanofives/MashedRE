## Mechanical description
- Signature `int FUN_004ebb00(short *out_tris, short *strip, int strip_len, int base_offset)`.
- Triangle-strip-to-triangle-list converter (with optional index offset). Two top-level paths based on `base_offset == 0`:
  - With `offset==0`: for each strip vertex `i in 0..strip_len-3`, emits the triangle (s[i], s[i+1], s[i+2]). Even-indexed iterations write in straight order, odd in reverse (winding flip per strip-rule). Skips degenerate triangles (any 2 equal indices).
  - With `offset!=0`: same but subtracts `offset` (cast to short) from every emitted index.
- Returns number of indices emitted (= 3 * non-degenerate-triangle-count).
