## Mechanical description
- Signature `int FUN_004e8a10(int geom)`. Builds the mesh-list for a geometry from its triangle/material/texture data.
- Returns `geom` if mesh-list already exists (`geom[0x54] != 0`).
- Calls FUN_004f0ae0(geom[0x10]) to create a fresh mesh-list with capacity for `numVerts` entries.
- Allocates three deduplication tables: material-pointer set, texture-pointer set, alpha-flag set (each `numMaterials * 4` bytes).
- For each triangle (`for iStack_18 in 0..numTriangles`):
  - Fetches triangle's vertex shorts (`puVar1 = &geom[0x2c] + i*8`) and material entry (`FUN_004f3bd0(matlist, tri.matIdx)`).

- Frees the three temp tables.
- Finalizes mesh-list via FUN_004f3b00(meshlist, has_native_flag). Stores result at `geom[0x54]`. On failure, frees mesh-list via FUN_004f0bd0.
