## Mechanical description
- Signature `undefined4 *FUN_004e9e40(stream, int *arena_top, int world, uint format_flags)`. **RpWorldSectorStreamRead**.
- Reads inner struct chunk; version-bounds-checks (0x35000..0x37002).
- Reads 44-byte body header (numTriangles, numVerts, etc.) into locals.
- Allocates sector from the world's resource arena: bumps `*arena_top` by `DAT_006189e4` (RpWorldSector class size), returns the old top. On NULL: emits 0x80000013 error.
- Initializes WorldSector header: `*sector = 0xffffffff` (root-marker), pastes bounding-box (6 floats from local_20..local_c), texcoord-stats (3 ushorts at +0x80/+0x82), zero out misc fields.
- Builds doubly-linked-list sentinels at sector[0xe..0x11] (sub-lists).
- If not native-only (format_flags has lighting bits) AND has numVerts/numTris:
  - Reads vertex array (numVerts * 0xc bytes) bump-allocated from arena.
  - Optional prelights (flag 0x10): reads numVerts * 4.
  - Optional vertex-colors (flag 0x8): reads numVerts * 4.
  - Per texcoord-set: reads numVerts * 8 bytes.
  - Reads triangle array (numTris * 8) bump-allocated.
- Registers sector with world, calls FUN_004d8000 (plugin-constructors), FUN_004e1b60 (plugin-stream-read walkers), FUN_004e1c90 (plugin-callback). On failure: branches on flag bit 1 of `world+3` to either FUN_004e4f70 (full world destroy) or FUN_004e5020 (sector-only destroy).
