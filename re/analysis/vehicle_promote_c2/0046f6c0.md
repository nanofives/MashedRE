## Purpose

Main per-tick wheel-track contact classification and position/orientation correction. Runs a 3-state machine per wheel (0=no contact, 1=light, 2=full), accumulates contact forces, applies position/orientation correction, aligns vehicle to track surface, and writes grounded-wheel count.

## Shape

**Signature:** `void VehicleWheelContactSolver(int param_1)`
- param_1 = substep flag (0 or 1)
- EDI = implicit vehicle struct pointer
**Returns:** void

**Reads (EDI-relative byte offsets):**
- `EDI+0x194` — wheel-0 spring load base
- `EDI+0x198` — wheel-0 contact state base (piVar9 stride 0x31 × 4 per wheel)
- `EDI+0x9AC` — wheel ring index (0 or 1)
- `EDI+0x9E0` — grounded-wheel count (read and written)
- `EDI+0x9F0` — no-contact-pending flag

**Writes (EDI-relative byte offsets):**
- `EDI+0x9E0` — grounded-wheel count (output)
- `EDI+0x9B0..+0x9B8` — linear velocity XYZ (friction impulse applied)
- `EDI+0x144..+0x14C` — angular impulse accumulator XYZ
- `iVar12 + 0x30..0x38` — wheel contact position correction (XYZ)

Where `iVar12 = EDI[0x9AC] * 0x40 + 0x928 + EDI` (current wheel-ring state block pointer).
