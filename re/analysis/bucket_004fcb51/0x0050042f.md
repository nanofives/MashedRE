## Mechanical description

**Whitespace + comment + line-continuation skipper** (`__fastcall(this)`).

The first phase of the lexer's "get next token" pipeline: advances `this->cursor` (`*this` = `param_1[0]`) past all skippable characters between meaningful tokens. Called as the first action from `FUN_00500adc` (the main lexer dispatch).

Returns `local_8` (= 1 if at least one `'\n'` was consumed, 0 otherwise) — this is the "we crossed a line boundary" flag the caller needs to gate `#`-at-start-of-line detection.

| First byte | Action |
