### Phase 2 — Substitution

Walks the macro definition body (`param_3` → `*(param_3+0xc)` chain). For each body-token-AST node:

- If token-class != 9 (identifier) AND no pending `#` or `##` mark: emit a copy of the token directly into the output via FUN_005107df.

- **`#@` charizing** (Microsoft extension): same path but with `local_10 = 1` instead — produces single-quoted char literal.
- **`##` token pasting** (post-substitution pass): walks the output chain, finds adjacent tokens separated by literal `##` token; concatenates their raw text via `FUN_004ff936` + manual dword/byte memcpy, then re-lexes via `FUN_004ffbc5` to produce a single new token replacing both.
