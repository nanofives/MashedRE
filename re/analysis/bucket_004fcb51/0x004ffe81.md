## Mechanical description

**Single/multi-char punctuator lexer** (`__thiscall(this, source_ptr, out_byte3)`).

Recognises 1-, 2-, or 3-character punctuator tokens starting at `param_2`. Writes up to three raw bytes into `param_3[0..2]` (zero-init the 4-byte slot first), and returns the number of source bytes consumed (1, 2, or 3).

### Recognised tokens

- **3-char**: `<<=`, `>>=`, `...` (only when the lookahead string is exactly `..` after the first `.`).
- **2-char digraphs** (`*(this+4)` end-pointer permitting):
  - `##`, `#@` — preprocessor token-paste / stringize-like glue.
  - `<<`, `>>`, `||`, `&&`, `::`, `++`, `--` (self-doubled).
  - `<=`, `>=`, `==`, `!=`, `/=`, `%=`, `+=`, `-=`, `*=`, `^=`, `|=` (assignment ops; the `cVar3 == '='` path with a sign-class guard on the first byte: `/`, `!`, `%..&`, `*..+`, `-`, `<..>`, `^`, `|`).
  - `->` (arrow).
- **1-char** fallback: any other single byte.

### Flow

The function uses a layered classifier:

1. Clear `param_3[0..3]`, store `*param_2` at `param_3[0]`.
2. If only 1 byte left in source: return 1.
3. **`#`-glue path**: if `cVar2 == '#'` AND `*(param_2+1) ∈ {'#','@'}` → emit 2-byte glue, return 2.
4. **Self-doubled path** (`cVar2 == *(param_2+1)`):
   - `'<'` or `'>'`: emit `<<`/`>>`. If next byte is `'='` (and end-pointer permits): emit `<<=`/`>>=`, return 3; else return 2.
   - `':'`, `'&'`, `'+'`, `'-'`, `'|'`, `'='` (range `< ';'` plus the trailing `<`, `=`, `>`, `|` cases): emit the doubled-op, return 2.
   - `'.'`: only acts if the third byte is also `'.'` → emit `...`, return 3; else return 1.
5. **`=` digraph path** (`cVar3 == '='`): first-byte must be one of `/ ! % & * + - < > = ^ |` (sign-bucketed: classes `<'0'` and `>='0'`). Emit `X=`, return 2.
6. **`->` path**: if `cVar2 == '-'` AND `cVar3 == '>'`: emit `->`, return 2.
7. Otherwise: return 1 (raw single-byte punctuator).

### Inferred token shape

Whoever consumes the returned `param_3[0..2]` does byte-compares to discriminate operator class — this is the punctuator slot of the main lexer's token-class 1 (single-char operator) output.
