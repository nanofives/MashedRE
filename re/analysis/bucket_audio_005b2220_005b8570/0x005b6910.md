# 0x005b6910 — FUN_005b6910 (C2 plate)

## Mechanical description
IMA ADPCM stereo decoder: two parallel predictor/step states for L and R, with interleaved 4-byte
data words (one dword per channel per group). Per-nibble step via 0x005b6b00.

- `undefined4 FUN_005b6910(uint *param_1, short *param_2, int param_3)` — `param_1`=ADPCM input,
  `param_2`=interleaved PCM output (L,R), `param_3`=block count.
- Per block: read two header dwords — L predictor `sVar2`/step `local_8 = (hdrL>>0x10)&0x7f`, R
  predictor `sVar4`/step `local_4 = (hdrR>>0x10)&0x7f`. If either step `> 0x58` → return 0.
- Emit seed L/R as the first stereo frame. Then 8 groups; each reads an L data dword and an R data
  dword and decodes their nibbles in lockstep:
  `FUN_005b6b00(nibbleL, &predL, &stepL)` and `FUN_005b6b00(nibbleR, &predR, &stepR)`, writing
  interleaved `psVar7[0]=L, psVar7[1]=R`. Group count 8, last group 7 (seed frame fills the slot).
- Returns 1 on success.
