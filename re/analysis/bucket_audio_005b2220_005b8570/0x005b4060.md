# 0x005b4060 — FUN_005b4060 (C2 plate)

## Mechanical description
Computes the output-buffer byte length for converting a buffer from format `param_1` to format
`param_2`, scaling by the ratios of sample-rate, bits-per-sample and channel-count, then rounding
up to the output sample-frame alignment. Format descriptors [[U-6910]]. Leaf (FP arithmetic +
ROUND intrinsic only).

- `int FUN_005b4060(int *param_1, uint *param_2)`.
- `uVar1 = (byte)param_2[3] >> 3` = output bytes-per-sample (`param_2[3]` low byte = bits).
- Inner scaled size = ROUND( (rate_out/rate_in) × (bits_out/bits_in) × (chan_out/chan_in) ×
  size_in ) where rate = `*param`, bits = `(byte)param[3]`/`*(byte*)(param+3)`,
  chan = `*(byte*)((int)param+0xd)`, size_in = `param_1[2]` (+0x8).
- Result rounded up to `uVar1` alignment: `(inner + uVar1 - 1) & ~(uVar1 - 1)`, then a final ROUND.
- Returns the byte length.
