- Gets device interface ptr from *unaff_ESI[0] (first field = IDirectInputDevice8*).
- Calls vtable[25] = Poll(device). If result < 0 (failed):

- Calls vtable[9] = GetDeviceState(device, 0x50, auStack_58) — reads 80 bytes DIJOYSTATE into local buffer.
- If GetDeviceState fails: returns immediately.
- Shifts history: slides 16-frame history windows for axis data (offsets 0xd0..0x111 and 0xa0..0xa1 in device struct).
- Zeroes current-frame fields: unaff_ESI[0xf2], [0xb3], [0xb2], [0xd3], [0xd2]; zeroes 0x20 bytes at [0x102].
- Button bitmap: same 8-per-iteration loop as FUN_00496100; for 0x11 (17) buttons; stores at unaff_ESI[0x102..0x103] (32 bits = 32 button slots).
- Axis (X/Y) raw → float:

  - At iStack_60: raw Y value, same scale, negated.
- POV hat (iStack_44 = DIJOYSTATE POV field):
