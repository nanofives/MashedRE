## Mechanical description

Reads current keyboard state from the DI keyboard device (DAT_00772fb8) and converts to a compact 32-byte bitmap in param_1.

- FUN_004b6480(param_1, 0x100): zeroes 256 bytes at the output bitmap.
- FUN_004b6520(acStack_110+0xc, 0x100): zeroes the 256-byte local DI keyboard buffer.
- Calls vtable[9] = GetDeviceState(DAT_00772fb8, 0x100, acStack_110+0xc).
- On failure: calls vtable[7] = Acquire(DAT_00772fb8), then retries GetDeviceState once with 0x100 bytes to a different local buffer (uStack_114).
- If either call succeeds: enters conversion loop.
- Bitmap conversion loop (iVar5=1..256, step=8):
  - For each group of 8 keys, processes 8 individual key bytes.
  - A key byte < 0 (high bit set = pressed) sets the corresponding bit in param_1.
  - Output bit position = key index (so bit k in the 32-byte output = DIK_k pressed).
- Returns 1.
