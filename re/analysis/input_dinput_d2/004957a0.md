- Computes stride: iVar1 = param_1 * 0x89 + param_2.
- If param_3 != null: *param_3 = *(int*)(&DAT_00772150 + iVar1*8); param_3[1] = same offset+4.

- Returns 1 on success, 0 on invalid indices.
