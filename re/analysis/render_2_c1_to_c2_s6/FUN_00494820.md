## Mechanical description
Probes and validates D3DX texture surface format. Has SEH frame.

### Setup
- Reads `iVar3 = *(int*)(param_2+0x44)` — pointer to D3DX texture descriptor.
- `iVar2 = *(int*)(iVar3+0x34)` → `param_1+0x164` (width? mip count?).
- `uVar1 = *(uint*)(iVar3+0x38)` → stored as abs value at `param_1+0x168`.
- `param_1+0x16c = ((param_1+0x164) + 1) * 3 & ~3` — pitch aligned.

### Power-of-two expansion
- Via SEH frame reads flags at `bStack_114 & 2`:
  - If `bStack_114 & 2 == 0`: use raw dimensions.
  - Else: expand width/height to next power of two ≥ original.
- Stores expanded values in `DAT_007719ec..DAT_007719f8`.

### D3DX surface creation
- Calls `(**(code**)(DAT_00771a0c+0x5c))` (DirectX create surface / `CreateImageSurface`) twice:
  - First call: probe call.
  - Second call: `(DAT_00771a0c, width, height, 1, 0, 0x16, 1, &DAT_00771a14, 0)` — create surface with format D3DFMT_A8R8G8B8 (0x16 = 22).
- If surface created: queries format via `(**(code**)(piVar7+0x30))(piVar7, &piStack_1ac)`.
- Format validation: only accepts `0x16` (D3DFMT_A8R8G8B8) or `0x19` (D3DFMT_X8R8G8B8).
  - On unsupported format: logs `"Texture is format we can't handle! Format = 0x%x"` via `FUN_00493f00`; releases surface; returns `-0x7FFBFDD6`.
  - On success: releases surface; returns 0.
- On create failure: logs `"Could not create the D3DX texture!  hr=0x%x"` via `FUN_00493f00`; returns HRESULT.
