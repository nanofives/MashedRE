the object's +0x294 flags word, maps it to a mode code written at +0x28c, and
(when the global guard 0x00636acc is clear) forwards the object's slot index
(+0x290) plus the computed sub-value to SlotFieldSet. Pure control-flow + field
writes; no loops, no allocation. No uncertainty markers — every constant and
offset is literal in the decompilation.
