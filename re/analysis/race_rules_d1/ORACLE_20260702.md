## Method

`DAT_007f0fd0` = rule under mode 10 Quick Race). Three `Interceptor.attach`
hooks (~60 calls/s total — far under the 1000/s hot-path limit):

| Hook | Original | Ported law being validated |
|------|----------|----------------------------|
| segment check | `FUN_00410d10` (0x00410d10) | `RuleEngine::SegmentCheck` (pre-blocks on entry state, alive-count tail on exit state) |
| result eval | `FUN_00410510` (0x00410510) | `RuleEngine::EvaluateResult` (+ `DAT_007f0fcc` p0-won side effect) |
| finish order | `FUN_004177b0` (0x004177b0) | `RuleEngine::UpdateFinishOrder` (entry order + exit metrics → predicted appends) |

Per call: read the port's documented input globals at `onEnter` (rule
0x007f0fd0, metric[4] 0x0089a880, score[4] 0x008a94e0, participants
0x008a94d0, finishOrder[4] 0x0089a870, timer 0x007f0fe4, collect
0x0063a5d0/d4, teams 0x0067ea64, motion 0x008815a0+i*0xd04+0x9f0, snapshot1
0x00899724, slot probe [0x005f2770]+0x34+i*4; getters `FUN_00443080`,
`FUN_0042f6a0`, `FUN_0046c7b0(i)` CALLED via NativeFunction), compute the
transcribed law in JS, compare with the original's return value (and
`DAT_007f0fcc` when concluded) at `onLeave`. No writes, no re-execution — the
original's own elimination/scoring side effects run untouched inside the call.

Edge forcing (CONTRIVED, C3-grade, same class as `--boost`): `--poke-lap`
writes the lap counter (row 0x008a9620+car*0x30c+0x28, U-8988) so the
ORIGINAL's own metric writer produces a finished metric next tick;
`--rule10-timer` seeds `DAT_007f0fe4`; `--poke-collect` writes the rule-5
counters (did NOT hold — zeroed continuously in a collectible-less context,
consistent with D-11056).

   per-round restart path: rule-8 run observed order `[0,-1,-1,-1]` reset to
   all `-1.0f` between `FUN_004177b0` calls across a round cycle (spawn wave

   `rulep_.finishOrder` per round. The rule-10 countdown is NOT re-seeded per
   round (3 back-to-back expiry rounds ran on a stale timer) — not mirrored.

   `FUN_00423b20(1)`) is **nonzero (1002/1003) during live quick-races**;
   with rule 9 forced, the original segment-ends on the FIRST tick via this

   challenge zeroes it at start is unknown. Standalone keeps feeding 0.

   Consistent with WS-G1's mode→rule derivation running per round.
