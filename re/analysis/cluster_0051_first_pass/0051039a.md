## Mechanical description

D3DX-style memory-mapped-file loader. Takes `param_1` (4-dword out struct), `param_2` (LPCWSTR path), `param_3` (wide-flag bool).

Steps:
1. If `param_2 == NULL` return `0x8876086c` (D3DERR_INVALIDDATA family).
2. If `param_3 != 0` (wide-string requested):
   - Call `GetVersionExA` (sized 0x94 = sizeof(OSVERSIONINFOA)).
   - If `dwPlatformId != 2` (not Windows NT family), call `WideCharToMultiByte(CP_UTF8=0xfde9, ...)` twice — first to query length, second to convert; force `param_3 = 0` and rebind `param_2` to stack buffer.
   - Else call `CreateFileW(path, GENERIC_READ, FILE_SHARE_READ, NULL, OPEN_EXISTING, FILE_FLAG_RANDOM_ACCESS=0x10000000, NULL)`.
3. Else (or fallthrough from Win9x path): `CreateFileA` with the same flags.
4. Store file handle to `param_1[0]`.
5. If handle != INVALID_HANDLE_VALUE:
   - `CreateFileMappingA(file, NULL, PAGE_READONLY=2, 0, 0, NULL)` → `param_1[1]`.
   - If mapping != INVALID_HANDLE_VALUE:
     - `GetFileSize(file, NULL)` → `param_1[3]`.
     - If size != INVALID_FILE_SIZE:
       - If size == 0: `param_1[2] = 0`, return 0.
       - Else `MapViewOfFile(mapping, FILE_MAP_READ=4, 0, 0, 0)` → `param_1[2]`; if non-null return 0.
6. On any failure path: `GetLastError()`, return `0x88760b59`.

Stack uses 0x94-byte `OSVERSIONINFOA` + small WCHAR→MBCS conversion buffer (`__chkstk` probe injected by compiler).
