## Mechanical description

Logs OS version information to `mashed.log` via `FUN_00496400` (debug printf).

**Sequence:**
1. Logs `"Operating System:\n"`.
2. Zero-initialises a 0x9c-byte `OSVERSIONINFOA` struct on stack.
3. Tries `GetVersionExA` with `dwOSVersionInfoSize = 0x9c` (extended struct). On failure retries with `0x94` (base struct). On both failures logs `"Unable to get OS version\n"` and returns.
4. Branches on `dwPlatformId`:
   - `0` (VER_PLATFORM_WIN32s): sets `DAT_00773920 = 7`, logs `"Microsoft Win32s "`.
   - `1` (VER_PLATFORM_WIN32_WINDOWS):
     - `dwMajorVersion == 4, dwMinorVersion == 0`: sets `DAT_00773920 = 2`, logs `"Microsoft Windows 95 "`, optional `"OSR2 "` suffix.
     - `dwMajorVersion == 4, dwMinorVersion == 10` (0xa): sets `DAT_00773920 = 3`, logs `"Microsoft Windows 98 "`, optional `DAT_005d028c` suffix.
     - `dwMajorVersion == 4, dwMinorVersion == 0x5a` (90): sets `DAT_00773920 = 4`, logs `"Microsoft Windows Me "`.
   - `2` (VER_PLATFORM_WIN32_NT):
     - `dwMajorVersion < 5`: sets `DAT_00773920 = 1`, logs `"Microsoft Windows NT "`.
     - `dwMajorVersion == 5, dwMinorVersion == 0`: sets `DAT_00773920 = 5`, logs `"Microsoft Windows 2000 "`.
     - `dwMajorVersion == 5, dwMinorVersion == 1`: sets `DAT_00773920 = 6`, logs `"Whistler "`.
     - NT path also queries registry `HKLM\SYSTEM\CurrentControlSet\Control\ProductOptions\ProductType` for Workstation/Server classification.
   - other: logs `"Unknown OS"`.
5. Logs `"\n"` at end.
