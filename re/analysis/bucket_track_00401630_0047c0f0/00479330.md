## Purpose

`Course::CreateFromDescription` — the master course asset loader. Given the course struct
`param_1`, the filename table `param_2` (fixed 0x40-byte string slots), and the descriptor
`param_3` (int array of counts/params), it loads, in order: texture dictionary (+0x105d0),
world BSP (+0x105d4), N collision BSPs (+0x10090), hoppy-collision BSPs (+0x100a0), particle
systems (+0x10330 group), AI data (+0x10080), UVA dicts (+0x10618), clumps (+0x103b0 ×0x40),
sky-shadow model (+0x105d8), fog/clip params, sky light atomics (+0x105e0/+0x105e4) or sky DFF
(+0x105dc), MTS (+0x10000 ×0x10), sky domes (+0x10210 ×4), PVS (+0x105ec/+0x105e8), splines
(+0x100b0 ×0x10), camera path (+0x105f0), general anims (+0x105b0 ×8), course objects
(+0x10110), then installs the post-load callback (`FUN_004790e0`) and sets `+0x10638`. Returns
0 on clump/spline failure, else 1.

## Shape

**Signature:** `undefined4 FUN_00479330(int param_1, char *param_2, int *param_3)`
— `param_1` course struct, `param_2` 0x40-stride filename table, `param_3` descriptor (counts at
`[1],[2],[4],[6],[7],[8],[0x1b],[0x35..]`; floats at `[0x76..0x7f]`).

**Returns:** `1` on success; `0` if a clump (+0x980 region) or spline (+0x1e00 region) fails.

**Body (0x00479330..0x00479e7d):** see decomp for the full per-resource load sequence with
its log strings; structure is a series of count-bounded loops over `param_2` filename slots,
each loading into the matching fixed offset on `param_1`. Fatal failures (world BSP, collision)
call `FUN_00478200(code)` (codes 0, 2).
