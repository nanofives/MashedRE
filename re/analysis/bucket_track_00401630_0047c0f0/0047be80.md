## Mechanical description

Per-triangle face-normal calculator. For each of `param_3` triangles, reads the three vertex
positions (12-byte stride, from the vertex base in `unaff_EBX`, indexed by the triangle's three
ushort indices), computes the cross product `(C-B) × (A-B)` — written component-wise — and
normalizes the result in place via `FUN_004c39b0`. Output written 3 floats per triangle.

## Shape

**Signature:** `void __fastcall FUN_0047be80(float *param_1, int param_2, int param_3)`
— `param_1` = output normal array (3 floats/tri), `param_2` = triangle index array base (ushort
indices, `+4` to skip a 4-byte header, stride 8 shorts), `param_3` = triangle count,
`unaff_EBX` = vertex position base (12 bytes/vertex). `__fastcall` (ECX/EDX + stack).

**Body (0x0047be80..0x0047bf6a):** per-triangle loop:
- load `A = vbase + idx0*0xc`, `B = vbase + idx_prev*0xc`, `C = vbase + idx_prev2*0xc`.
- `n.x = (C.z-B.z)*(A.y-B.y) - (C.y-B.y)*(A.z-B.z)` and the two cyclic permutations.
- `FUN_004c39b0(param_1, param_1)` (normalize); advance `param_1 += 3`, `param_2 += 4 dwords`.
