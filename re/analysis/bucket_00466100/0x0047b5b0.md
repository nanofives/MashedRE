## Mechanical description
- Same bytecode 3-byte read pattern as 0x47b480.
- Splits first byte into `outer = byte0 >> 3` (player or super-group, 0..7) and `inner = byte0 & 7` (sub-group, 0..7).
- Computes record base: `&DAT_0086ee28 + outer*0x944 + inner*0x117` (outer stride 0x944=2372B; inner stride 0x117=279B).
- Reads current count at base + 0xb (= dword index 11). Writes byte2 to `base + count + 3` and increments count.
