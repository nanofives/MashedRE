## Mechanical description

`RtFSManagerRegister`-equivalent. Wires up an FS record + initializes its
slot array + appends to the global FS list.

1. Check FUN_00550400 (registration prerequisite gate). If 0, return 2.
2. Check FUN_00550520 (name not already registered). If non-zero (already
   exists), return 6.
3. Copy `param_3` (name) into `&fs[0x50]` via `vtable[+0xcc]` (strcpy).
4. Set `fs[1] = param_2` (slot count), `fs[4] = 0` ("default ops flags"?).
5. For each slot 0..param_2-1: get slot via `fs_vtable[+0x14](fs, idx)`,
   set slot+0x20=1 (free state), slot+0x18 = fs[4] (carrying default).
6. Lookup `param_4` (scheme/extension name) via FUN_00550750.
   - If NOT already in list: alloc a buffer of `strlen(param_4)+1` via
     `vtable[+0x108]`, strcpy `param_4` in. Store the buffer at fs+0xc.
   - Set fs+8=2 (state=registered), fs+0x20=0, fs+0x24=0. Return 1.
   - If ALREADY in list: return 7.

Return codes: 1=success, 2=prereq-fail, 6=name-collision, 7=scheme-collision.
