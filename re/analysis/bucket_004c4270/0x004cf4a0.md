## Mechanical description

- Signature `RwTexture* f(RwTexture* tex, RwStream* stream)`.
- Computes total size via the same recipe as 0x004cf460
  (`FUN_004d8770(tex+0x10) + FUN_004d8770(tex+0x30) + 0x34 + plugin_size`).
- Writes outer chunk header: `FUN_004cc580(stream, 6, total_size,
  0x37002, 10)` — RW chunk-write with type=6 (TEXTURE), chunk-id
  0x37002, version=10 (which decodes to RW 3.5+).
- Writes inner header: `FUN_004cc580(stream, 1, 4, 0x37002, 10)` —
  STRUCT subchunk (type=1), size=4.
- Builds packed flags DWORD:
  - bit 16: `1` if (`tex->raster != NULL && (raster->flags & 0x10) == 0`)
    else `0` — "has-mipmaps-or-not" inverse.
  - bits 0..15: `tex[0x14] & 0xffff` (the texture filter/address mode mask).
- Writes the packed 4-byte payload via `FUN_004cbe80(stream, &flags, 4)`.
- Writes the name string via `FUN_004d87a0(tex+0x10, stream)` —
  string-stream-write (companion to FUN_004d8770).
- Writes the mask-name string via `FUN_004d87a0(tex+0x30, stream)`.
- Writes plugin-registered data via `FUN_004e1d20(&DAT_00618138,
  stream, tex)` — `_rwPluginRegistryWriteDataChunks`.
- Returns `tex` on full success, 0 on any-step failure.
