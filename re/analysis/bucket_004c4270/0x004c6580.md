  int mip_count, uint base_log2_size)`.
- Allocates 16KB stack alloca'd palette work-buffer (`undefined1

- Fast-path (all-equal): if every mip's palette pointer (read via

  (RwImageAllocatePixels), runs `FUN_004da4f0` (Floyd-Steinberg
  remap), sets the new mip's palette pointer to `dest_palette`, and
  replaces the old mip in the array.

  matches `param_2` (the original — keep it alive).
- Finalizes via `FUN_004daaf0` (palette tear-down).
- Mechanical shape: **mipmap palette unification** — when a texture
  has differently-paletted mip levels, builds a single shared palette
  and remaps every mip to that palette. Standard step in RwTextureRead
  for paletted-format mip chains.
