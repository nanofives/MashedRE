## Mechanical description

- Signature `int f(int new_pipeline_or_null, u32 prim_type)`.
- Dispatch by `prim_type` (1..6); writes (or restores) the per-prim
  pipeline slot at base `DAT_00911d00 + DAT_007d3ff8`:
  - 1 (LineList) -> base+0x10
  - 2 (LineStrip) -> base+0x14
  - 3 (TriangleList) -> base+0x4
  - 4 (TriangleFan) -> base+0xc
  - 5 (TriangleStrip) -> base+0x8
  - 6 (PointList? or other) -> base+0x18
  - default: raises RwError 0x25 (same code as 0x004cd170 for bad
    prim type).
- Two halves:
  - `new_pipeline != 0`: write into the per-prim slot, return it.
  - `new_pipeline == 0`: restore from the "saved default" slot:
    - 1 -> base+0x2c
    - 2 -> base+0x30
    - 3 -> base+0x20
    - 4 -> base+0x28
    - 5 -> base+0x24
    - 6 -> base+0x34
    If the saved-default is 0, also writes 0 into the per-prim slot.
- Mechanical shape: **RxPipelineSetCurrent(prim_type, pipeline)** /
  **Reset** — the full version of 0x004cd500's generic helper, with
  per-primtype dispatch.
