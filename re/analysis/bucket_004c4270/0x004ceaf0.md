## Mechanical description

- Signature `int f(RwImage* dst, RwImage* src)`.
- **Palette copy** (if both have palette ptrs at `+0x18` and src->depth < 9):
  - `palette_size = 1 << src->depth` (entries), each 4 bytes.
  - Copies via 4-byte-then-remainder dword copy (`uVar3 >> 2` then
    `uVar3 & 3`). The 4-byte trailing loop is `for (iVar4 = 0; ...)`
    — Ghidra rendered zero-iteration; this is a dead-tail because
    `palette_size * 4` is always a multiple of 4.
- **Pixel copy**: per-row, `width * ((depth+7)/8)` bytes:
  - Outer iterates over `dst->height` rows.
  - Inner copies `((depth+7)>>3) * width` bytes per row.
  - Source stride from `src + 0x10`, dest stride from `dst + 0x10`.
- Always returns 1.
- Mechanical shape: **RwImageCopy fast-path when src->depth == dst->depth**.
  Companion to 0x004ceba0 (cross-depth converter).
