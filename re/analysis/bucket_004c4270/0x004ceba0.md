## Mechanical description

- Signature `int f(RwImage* dst, RwImage* src)`.
- Reads `src->palette` (slot +0x18) for palette lookup. Builds
  `uVar3 = (src->depth << 8) | dst->depth`.
- Switch on the packed depth-pair `uVar3`:
  - `0x808` (8->8): no-op, returns 1
  - `0x404` (4->4): no-op, returns 1
  - `0x408` (4->8): direct pixel copy, returns 1
  - `0x420` (4->32): falls through to palette expansion
  - `0x820` (8->32): falls through to palette expansion
  - `0x2020` (32->32): no-op, returns 1
  - default: raises RwError `0x80000009` ("depth unsupported"),
    returns 0
- **Palette expansion** (`0x420` / `0x820`):
  - Per row, per column, reads 1 byte from src, looks up
    `palette[byte*4]` as a 4-byte DWORD (full RGBA pixel), writes
    to dst.
  - Strides from `src + 0x10` and `dst + 0x10`.
- Always returns 1 on success, 0 on error.
- Mechanical shape: **RwImageCopy cross-depth converter**. Specifically
  handles palette -> true-color expansion.
