## Mechanical description

- Signature `RwImage* f(const char* color_path, const char* mask_path)`.
- Reads the color image via `FUN_004ce390(color_path)` — the
  format-dispatch-by-extension above. Returns NULL if color read fails.
- If `mask_path == NULL` OR `mask_path[0] == '\\0'`: returns the color
  image directly (no mask).
- Else: reads the mask image via the same path. If mask read fails,
  destroys the color image (freelist + list-remove dance identical
  to RwImageDestroy/RwImageFreePixels) and returns NULL.
- Calls `FUN_004cde50(mask)` (RwImageMakeMask — same as plate
  0x004cde50: forces alpha = max(R,G,B)).
- Calls `FUN_004cdf20(color, mask)` (RwImageCopy — same as plate
  0x004cdf20: copies the alpha-only mask into the destination).
- On either step fail: full cleanup of both images via the destroy
  sequence; returns NULL.
- On success: destroys the mask image (color is now masked), returns
  the color image.
- Mechanical shape: **RwImageReadMaskedImage** — composite read where
  a separate mask file provides the alpha channel for a base color image.
