## Mechanical description

- Signature `bool f(u32 a, RwRaster* raster, u32 flags)`.
- Guard: `raster[1] != 0` (raster already locked) -> RwError
  `0x8000000e` ("raster busy"), return 0.
- Walks the raster parent chain (`piVar9 = *raster; while (piVar5 = piVar9,
  piVar9 = *piVar5, piVar5 != piVar9)`) to find the "root" raster
  (the parent atlas / chain head).
- Computes lock-flags based on `flags & 1` ("read-only"): 0x800
  (RWRASTERLOCK_READ?) or 0x8810 (READ|WRITE|NOSYSMEM-like).
- Switch on lower 3 bits of `raster[8]` (raster type):
  - **0/4** (texture / texture-array): reads byte tag at
    plugin-offset +9 of root raster. Dispatches plugin-vtable +0x48
    (Texture-CreateSurface helper, same as plate 0x004cafd0). On
    success, calls the device-vtable +0x34 (CreateAdditionalSwapChain
    OR CreateOffscreenPlainSurface) to upload the texel pointer at
    `&raster->[plugin_offset + 0x10]`. Sets raster bit `0x22` flags
    on success.
  - **2/5** (volume / cube): plugin-vtable +0x48 dispatch. If type==5,
    must succeed (returns 0 on fail). Calls device-vtable +0x90
    (CreateRenderTarget / CreateDepthStencilSurface) with 6-arg form,
    then vtable +0x80 (UpdateSurface) if `flags & 2` set.
    Optionally calls Release (vtable +0x8) on the plugin obj if
    type==5.
  - **default**: RwError `0x8000000e`, return 0.
- Post-success bookkeeping:
  - Stores `raster[1] = raster[+0x14]` (the locked-surface pointer
    location at plugin-offset 0x14).
  - Copies `raster[3], raster[4]` into `raster[10], raster[11]`
    (width/height "saved-original").
  - Right-shifts `raster[3]` and `raster[4]` by mip-level (the
    byte at `puVar11`); clamps to >= 1.
  - Sets `raster[6] = raster[plugin_offset + 0x10]` (pixel-data ptr).
  - Stores mip level at plugin-offset +0xb.
  - `*piStack_1c = raster[1]` (write-back to the parent slot).
- Mechanical shape: **RwRasterLock** (or **RpD3D9RasterCreateNative**)
  — the function that resolves a raster's native D3D9 surface,
  locks it for CPU access, and back-fills the raster's
  CPU-readable mirror fields.
