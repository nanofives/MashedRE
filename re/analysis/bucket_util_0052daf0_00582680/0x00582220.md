`s_qh_triangulate_link__relink_old_f_...`, `s_qhull_error__qh_triangulate_link_...`.

Relinks neighbor references when a degenerate facet is replaced: given old facet
`param_1`/neighbor `param_2` and new facet `param_3`/neighbor `param_4`, checks the
mutual neighbor membership (`qh_setin` both ways). If both reference each other it
adds a MERGE marker (`qh_setreplace(...,8,0)`); otherwise errors via
`qh_errexit(5,...)` if the link is broken. Then swaps the neighbor-set entries
(`qh_setreplace FUN_005833c0`) so `param_4`'s neighbors point to `param_3` and
`param_2`'s point to `param_1`.
