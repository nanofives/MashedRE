## Mechanical description

**Qhull 2002.1 `qh_nextridge3d`** — returns the next ridge around a 3-d facet.

Given the current ridge `param_1`, facet `param_2`, and optional out-vertex
`param_3`: determines this ridge's "other" vertex `iVar4` (the vertex shared in the
3-d ring, picked from `*param_1+4`/`+8` per which side `param_2` is on). Scans the
facet's ridge set (`param_2+0x24`) for the ridge whose corresponding vertex equals
`iVar4` (i.e. the adjacent ridge in the ring); on match, optionally writes the
shared vertex to `*param_3` and returns that ridge. Returns NULL if none.
