## Mechanical description

**Qhull 2002.1 `qh_pointvertex`** — builds the point→vertex lookup table.

Allocates a point-indexed array of size `qh_setsize(DAT_00914664) + DAT_00914024`,
zeroes it. Walks the vertex list `DAT_009145c4` (next at +0); for each vertex,
indexes by its point id (`qh_pointid(vertex[2])`) and stores the vertex pointer in
that slot. Negative ids → internal warning; out-of-range → internal error +
`qh_errexit(5,...)`. Returns the array. (Structurally identical to `qh_pointfacet`
but over the vertex list.)
