## Mechanical description

**Qhull 2002.1 `qh_facet3vertex`** — confirmed by strings
`s_qhull_internal_error__qh_facet3v_...` (two variants).

Returns an ordered vertex set for a 3-d facet `param_1`. Allocates a result set
sized `qh_setsize(facet+0x20)`. If the facet is simplicial (`+0x3c & 0x2000`):
asserts dim==3, then appends the three vertices from the vertex set (with an
orientation-dependent insert at index 0 when the `0x1000` toporient flag differs).
Otherwise walks the facet's ridges in order via `FUN_00581170` (qh_nextridge3d),
appending each ridge's shared vertex (`local_8`); errors out via `qh_errexit(5,...)`
if the ridge ring is inconsistent or the count mismatches. Returns the vertex set.
