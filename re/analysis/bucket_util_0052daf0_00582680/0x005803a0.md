(Uses `qh_newvertex` inline; shares the `s_qhull_input_error__more_than__d_v_...`
and `s_qh_newvertex__vertex_p_d__v_d__c_...` strings, qhull memory/set API.)

Chooses the initial simplex vertices from the point set `param_2`:
- For the random/`Qr` path, draws indices via `qh_rand` (`FUN_00594c70` +
  `__ftol`), wrapping with the extra-points list.
- For high dimension (`dim > 7`) builds a max-simplex via `qh_maxsimplex`
  (`FUN_005940e0`) over a candidate set, appending vertices that increase the
  spanned volume; for low dimension uses the precomputed extremes.
- Finally allocates a vertex object (`qh_memalloc` 0x18, zero-init, id bump with
  `0xffffff` overflow check) for each chosen point and adds it to the result set
  `local_8`. Returns the vertex set.
