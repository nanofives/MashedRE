## Mechanical description

**Qhull 2002.1 `qh_triangulate`** — confirmed by ~10 strings
`s_qh_triangulate__triangulate_non__...`, `s_qh_triangulate__delete_null_face_...`,
`s_qh_triangulate__update_neighbor_l_...`, `s_qh_triangulate__identify_degener_...`,
`s_qhull_error__qh_triangulate___ri_...`, etc.

Triangulates all non-simplicial facets (no-op when dim==2). Switches centers to
centrums (`qh_clearcenters`/`FUN_005824a0`), then: (1) triangulates each eligible
facet via `qh_triangulate_facet` (FUN_00581ff0) into a fan; (2) deletes null/
degenerate facets, relinking mirror neighbors via `qh_triangulate_mirror`
(FUN_005822f0) over a temp set `DAT_0091465c`; (3) updates neighbor lists
(`qh_resetlists`); (4) identifies degenerate "tricoplanar" facets, replacing
representatives and deleting redundant ones via `qh_delfacet` (FUN_0059b7b0),
decrementing `DAT_009145b0`. Restores the `Qt`/good flags and re-checks via
`qh_checkpolygon` (FUN_0057e820) if requested. Heavily verbose-logged.
