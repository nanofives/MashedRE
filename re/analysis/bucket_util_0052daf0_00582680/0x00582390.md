Builds the intersection of two **sorted** vertex sets `*param_1` and `param_2`
(both ordered by descending vertex id `vertex+0x14 & 0xffffff`). Allocates a result
set of capacity `dim-1` (`qh_setnew(DAT_0091401c-1)`), then merge-walks both sets:
when ids are equal, appends the vertex (`qh_setappend`) and advances both; otherwise
advances the cursor whose id is larger. Frees the old `*param_1` (`qh_settempfree`),
stores the result, and registers it as a temp set (`qh_settemp` FUN_00583c50).
