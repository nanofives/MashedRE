## Mechanical description

**Qhull 2002.1 `qh_setappend`** — the canonical Qhull set-append primitive.

Appends element `param_2` to the set `*param_1` (no-op if `param_2 == 0`). The set
layout: `set[0]` = maxsize; the count slot is at `set[maxsize+1]`; elements occupy
`set[0..count-1]` with a trailing NULL terminator. If the count slot is 0 (full),
grows the set first (`qh_setlarger` FUN_00582f70) and re-reads. Then writes
`param_2` at the current end, bumps the count, and re-writes the NULL terminator
one slot further.
