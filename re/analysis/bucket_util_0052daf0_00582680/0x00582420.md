## Mechanical description

**Qhull 2002.1 `qh_vertexintersect_new`-style** — returns a NEW set of the
intersection of two sorted vertex sets `param_1` and `param_2` (cf. the in-place
variant FUN_00582390).

Allocates a result set of capacity `dim-1` (`qh_setnew(DAT_0091401c-1)`), merge-walks
both sets by descending vertex id (`vertex+0x14 & 0xffffff`): on equal ids appends
the vertex (`qh_setappend`) and advances both; else advances the larger-id cursor.
Returns the new set when either input is exhausted.
