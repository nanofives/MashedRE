# Workflow (Opus 1M - bulk-coherent per session)

time. Opus 1M context keeps all 56 functions' decomp + cross-refs in
scope simultaneously - exploit this for cross-function naming (recurring
callees, shared globals, repeated string patterns).

  1. mcp__ghidra__function_at <rva>     (or listing_code_unit_at if no fn)
  2. mcp__ghidra__decomp_function       (the body)
  3. Write re/analysis/bucket_00489450/<rva>.md with frontmatter:
       ---
       rva: 0x<rva>
       name_in_ghidra: <name returned by function_at>
       size_bytes: <from function_at>
       confidence: C1
       callees_depth1: [<list>]
       callers_noted: [<list>]
       opened_in_slot: Mashed_pool0 (or actual slot if fallback)
       session_date: 2026-05-20
       ---
       ## Mechanical description
       - <bullet 1: what the function reads/computes/writes>
       - <bullet 2: ...>
       ## Constants
       | Address | Value | Note |
       | ...     | ...   | ...  |
       ## Uncertainties
       <none | [UNCERTAIN] markers with paths to resolution>
       ## Stubs encountered
