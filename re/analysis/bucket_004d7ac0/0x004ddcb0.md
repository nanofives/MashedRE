## Mechanical description
- Clamps and adjusts (width, height) to satisfy D3D9 device caps.
- Clamps `*param_1` to `DAT_00911ff8` (D3DCAPS9::MaxTextureWidth) and `*param_2` to `DAT_00911ffc` (MaxTextureHeight).
- If `param_3 & 7 == 0` or `(param_3 >> 8) >= 0` (sign-bit check), and `_DAT_00911fdc & 2` is set with `_DAT_00911fdc & 0x100` cleared: rounds up to next power-of-two (counts leading bits, then `1 << count`).
- Else if `_DAT_00911fdc & 2`: same power-of-2 rounding (different gate).
- Finally if `_DAT_00911fdc & 0x20`: forces width=height by snapping the larger to the smaller (square-only caps).
